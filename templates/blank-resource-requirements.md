# 【对象名】资源需求（落地稿 · 空白模板）

> **这是"落地方向"模板**——设计锁定**后**，给美术 / 程序的**生产交付清单**:逐资产、逐帧、逐特效、逐程序需求，照着就能产出。
> 跟"审阅方向"（[`blank-design-doc.html`](blank-design-doc.html)）分工:审阅稿讲"这设计好不好"、含资源**概览**；本落地稿讲"具体做哪些、什么规格"、是**生产级明细**。两者一概览一明细，不重复。
>
> **怎么用**:① 复制本文件 → 改成 `output/characters/<对象名>-资源需求.md` 或 `output/bosses/<对象名>-资源需求.md`；② 删掉示例行、按你的设计填；③ 帧数按**你项目的帧率**（本模板示例用 60fps）；④ 资产命名按**你项目的命名规范**（见 [conventions](../conventions/TEMPLATE.md)）。
> **要团队 xlsx 格式**:填完本 md 后，用 [`scripts/`](../scripts/) 的脚本 inspect 你团队模板 → 生成对应 xlsx（仅 Windows + Excel/WPS）。
