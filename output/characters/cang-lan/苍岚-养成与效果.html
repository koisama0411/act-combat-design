      壹 · 设计说明

      本稿演示「技能附带效果 + 命座解锁效果 + 养成物效果 + 属性投放 + 数值建模验证」如何落到一个具体角色。

      🧪 跨语境说明：示例角色苍岚的 canonical 项目是买断制无命座的《孤刃行》。本稿把她移植到 gacha 示例项目《星澜纪》下，专门演示 gacha 的命座 / 养成物 / 属性投放设计 + 数值验证；与买断制原版（苍岚-战斗设计.md）是两个语境，不要混读。

      角色基础机制：势 / 一闪 / 凝势 沿用原版（基础动作 / 招式明细见角色主设计稿），本稿只展开「效果 + 养成 + 数值」三层。

      贰 · 技能附带效果

      任何项目通用 —— 每招挂什么 buff / debuff / 标记 / 状态。

      招式附带效果类型
      普攻 4 段末段击退 + 自身回 1「势」控制 + 资源生成
      完美闪避自身「残影」（可接反击）+ 回 1 势自 buff + 资源
      闪避反击对敌浮空控制
      凝势（核心特殊技）自身「止水」减伤 20% + 势 stack（每层 −移速）stacking buff（带权衡）
      斩势 1–3 层对敌削韧削韧 debuff
      破势·一闪（4 层）大削韧 + 穿透 + 对破韧 / 倒地目标处决标记条件处决
      弹反成功自身回 1 势 + 对敌硬直资源 + 控制
      终结技·奔雷自身霸体 + AOE 雷光自 buff + 群伤

      叁 · 命座 C0–C6 解锁效果

      按命座 / 星魂 5 段付费哲学：C0 必保完整 · C2 质变拐点 · C6 机制变质。

      命座效果段位定位
      C0势 / 一闪核心循环完整必保完整
      C1凝势回势 +15%数值小补
      C2处决条件放宽：「易伤窗内」目标也可处决质变拐点（解锁新触发）
      C3一闪 / 斩势 +2 级数值
      C4完美闪避 / 弹反回 2 势（原 1）循环提速（效果改）
      C5终结技 +2 级数值
      C6满势一闪后入「奔雷态」3s：普攻附雷光范围斩 + 凝势瞬满鲸鱼勋章（机制变质）

      伍 · 属性投放 · 资源「势」· 操作循环

      怎么堆词条、势怎么转、循环怎么打 —— 落地 / 养成向。

      属性投放（输出 · 单体）

      词条优先级说明
      暴击率 / 暴击伤害★★★（最高）双爆，目标比例 CR : CD ≈ 1 : 2（如 65% / 130%）
      攻击力 %★★★主词条主力，乘区直乘
      充能效率★★够循环点出终结技即可，过量浪费
      击破 / 减韧★（次要）苍岚靠一闪削韧已够，不主堆击破赛道
      增伤 %—（不主堆）她无自带增伤，走暴击赛道；硬堆增伤会踩「加法增伤区稀释」天花板（见「陆」边际）

      → 一句话：暴击赛道 + 攻击力，不碰增伤（避免稀释），充能够用即可。

      为什么 CR : CD = 1 : 2：暴击乘区 ≈ 1 + 暴击率 × 暴击伤害；双爆词条总量固定时，CR : CD = 1 : 2 令 CR × CD 期望最大。算例（同词条预算）：65% / 130% → 1 + 0.65 × 1.30 = 1.845；90% / 80% → 1 + 0.90 × 0.80 = 1.72 —— 前者更高，故按 1 : 2 堆。

      资源「势」经济（0–4 层）

      动作势变化
      普攻 4 段一套+1
      完美闪避 / 弹反成功+1（C4 后 +2）
      斩势（1–3 层）即时消耗，威力随层数
      破势·一闪（4 层）消耗 4，处决

      势只在释放 / 切人时清空（不随时间衰减）。张力：凝势每层 −移速，蓄越满越脆 → 「再蓄一层博伤害 vs 现在放保安全」。

      操作循环

      • 短轴：普攻 4 段（回 1 势）→ 凝势补到 2–3 层 → 看破绽放斩势
      • 长轴：弹反 / 完美闪避攒势 + 普攻铺垫 → Boss 大破绽 → 凝满 4 层 → 破势·一闪处决 → 终结技「奔雷」收尾
      • 心法：用防御操作（闪 / 弹）反哺进攻资源；在 Boss 破绽窗精准塞一记满势一闪
      ▲ 短轴日常铺垫；长轴在 Boss 破绽窗凑满 4 势打出破势·一闪处决，终结技收尾。金框 = 核心爆发点。

      陆 · 数值建模验证 + 边际效应

      用《星澜纪》乘区式（技能 × 增伤 × 暴击，守方乘区归一化为 1）跑 C0→C6，并看几处边际。本章数值为相对指数 / 倍率（已归一化），绝对伤害需代入项目真实面板复算。

      命座养成梯度

      命座C0C1C2C3C4C5C6
      累计指数100105118125134142195
      逐级增量—+5%+12%+6%+7%+6%+37%
      ▲ 命座养成曲线（金点 = 质变跳点 C2 / C6）。
      怎么算：每命座效果折算成对「循环 DPS」的增益 %，在 C0 = 100 基准上累乘成累计指数（乘区式 技能 × 增伤 × 暴击，守方乘区归一化为 1）。
      为什么这么算：C2 / C6 做质变跳点（付费拐点 + 鲸鱼勋章），其余平滑补，符合 5 段付费哲学。
      算的过程：C1 回势 +15% → 循环约 +5% → 105；C2 处决放宽 → 均摊约 +12% → 118；C3 技能 +2 级 → +6% → 125；C4 回势 ×2 → +7% → 134；C5 技能 +2 级 → +6% → 142；C6 奔雷态 → +37% → 195。
      结果：195 ÷ 100 = 满命 +95%，落 +80~120% 健康区。

      边际效应

      维度边际表现（示例）设计意图
      势层 → 斩势威力1 / 2 / 3 层 ≈ 1.0× / 1.4× / 1.9×；4 层一闪 ≈ 3.5× + 处决（边际跳升）4 层质变 → 鼓励「蓄满」博弈
      凝势 → 移速每层 −移速 ≈ 15% / 30% / 50% / 70%，边际递增的机动惩罚平衡「蓄满」风险：越肥越脆
      增伤词条加法 bucket，自拐边际 ≈ S ÷ (1+面板)；苍岚无自拐 → 不踩稀释走暴击赛道规避增伤天花板
      命座C0→C6 +95%，C2 / C6 质变跳点（见上图）付费哲学：平滑补 + 拐点放量
      ▲ 势越高 → 斩势/一闪倍率越高（蓝柱；4 势一闪 3.5× 质变，金柱），但凝势剩余移速越低（绿虚线 85%→30%）。这就是「蓄越满、越强、也越脆」的核心张力。
      怎么算 · 过程：斩势倍率按势层近线性偏上 —— 1 势 1.0×（基准），每层约 +0.45×，至 3 势 1.9×；4 势改走破势·一闪，跳到 3.5× 且附处决（质变、非线性）。移速惩罚每层递增 −15 / −15 / −20 / −20% → 剩 85 / 70 / 50 / 30%。
      结果：4 势伤害 ≈ 1 势斩势的 3.5 倍（基准为 1 势斩势 = 1.0；换算成 vs 普攻段约 8×，对应 §五「爆发」）、却只剩 30% 移速 —— 收益与风险同时见顶，逼出「再蓄一层 vs 现在放」的决策。

      § 五健康自检

      维度基线苍岚判
      满命总梯度+80~120%+95%✅
      逐级累计严格递增100 < 105 < … < 195✅
      C0 完整必须核心循环 C0 即完整✅
      爆发5~15×一闪 ≈ 8× 普攻段✅
      增伤区×≤1.5≈1.1（靠暴击）✅
      暴击CR:CD 1:2 不溢出遵守✅
      养成物套装4pc ≈ 质变命座量级「止水」4pc 条件 +40%✅
      PVP 满命 / 0 命≤1.8×原始梯度 1.95× 逼近上限；HP/防御不随命座涨，有效战力比 ≈1.6×⚠️ 临界
      场景梯度（群伤 C 单体）40~55%苍岚是单体角色N/A

      PVP 比怎么算：原始伤害梯度 = 累计指数比 195 ÷ 100 = 1.95×；但 PVP 战力还含 HP / 防御（不随命座涨），把伤害提升摊进总战力后有效比 ≈ 1.6×（示例估算）。1.95× 看着逼近 1.8× 上限、有效 1.6× 又在区内 —— 正因临界，必须用项目真实 PVP 公式跑，别用简化式拍板。

      🔎 本演示暴露的方法论价值：gacha + PVP 语境下，§ 五的 PVP 阈值主动拦下一个临界点（满命伤害梯度逼近 1.8× 上限），提示必须用项目真实 PVP 公式复核 —— 正是「用真实公式跑、别用简化式」的用例。

      柒 · 最终总结

      一句话：静如止水，动如奔雷；蓄得越满，一闪越狠，但也越脆。

      苍岚把「蓄力—释放」做成一套可主动管理的节奏资源（势），效果 / 养成 / 数值三层闭环：

      · 效果层 —— 势 / 一闪 / 止水：普攻·闪避·弹反生成势，斩势·破势一闪消耗，凝势减伤但减速；
      · 养成层 —— 命座 C0–C6 沿付费哲学平滑补 + C2 / C6 质变跳点，养成物套装强化流派；
      · 数值层 —— 暴击赛道（不碰增伤、避免稀释）+ 势层 / 凝势的递增边际 + 命座 +95% 健康梯度，经 §十 建模验证落在健康区。

      唯一待办：PVP 满命 / 0 命比临界（1.95× 原始梯度逼近 1.8× 上限），需用项目真实 PVP 公式复核 —— 这正是方法论「用真实公式跑、别用简化式」的价值所在。
