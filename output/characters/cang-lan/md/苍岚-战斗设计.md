## Sheet 1 · 战斗玩法

### 基本信息
| 项 | 值 |
|---|---|
| 角色名称 | 苍岚 |
| 资产名 | `CangLan` |
| 武器型 | 长柄（薙刀） |
| 队伍定位 | 输出 |
| 上手难度 | ★★（中） |
| 蓝本 / 形象 | 浪迹的青灰长衣女武者，背负一柄过身长的薙刀；静如止水、动如奔雷 |

### 设计思路
苍岚的核心是把"**蓄力—释放**"做成一套**可被玩家主动管理的节奏资源**，而不是一次性的长按重击。她长按特殊技进入「凝势」，随时间积攒「**势**」层（0–4），松手按当前层数释放对应威力的斩击——**层数越高越强，但凝势时移动变慢、且占用时间**，于是玩家永远在"再蓄一层博伤害"和"现在放保安全"之间做决策。满 4 层解锁「破势·一闪」居合，是她的爽点与处决手段。

这套设计让一个上手不难的近战角色，拥有**很高的操作上限**：高手会用普攻 / 完美闪避 / 弹反穿插回势，在 Boss 的破绽窗里精准塞进一记满势一闪。

### 核心记忆点
1. **「势」节奏管理**——不是无脑长按，而是"蓄到几层放"的实时博弈。
2. **满势一闪 = 处决**——4 层「破势·一闪」命中破韧目标直接处决倒地，高风险高回报。
3. **闪避 / 弹反回势**——防御性操作反哺进攻资源，鼓励贴身博弈而非苟着蓄力。

### 主题风格
"止水与奔雷"——凝势时极静（横刀垂目、青雾流转），释放时极烈（一闪而过、雷光炸裂）。视觉上靠"静—动"的强反差建立辨识度。

### 类似玩法（业界参考）
- 只狼·居合突刺的"蓄—放"张力
- 战神·利维坦之斧的资源 / 节奏管理感
- （以上为节奏感参考，机制为原创）

### 推荐配队（示例项目 = 单角色主导 + 可选 3 人小队）
- **苍岚（主控出伤）+ 一名坦克（拉仇恨，给她站桩凝势的安全窗）+ 一名支援（切人联动补伤 / 上易伤）**。
- 核心出伤点 = 苍岚的满势一闪；队友的职责是**制造"能安全蓄到 4 层"的窗口**。

### 被动机制 ·「止水」
- 凝势（蓄力）期间受到伤害减免 20%——鼓励"敢在敌人面前蓄力"。
- 但势每升一层，凝势时移动速度递减（4 层时几乎站桩）——蓄得越满、越脆、越需要队友 / 走位保护，形成张力。

### 养成提升（示例项目 = 买断制技能树，无抽卡命座）
技能树三条分支，玩家按流派点：
1. **势之道**：普攻回势加快 / 完美闪避额外回势（强化循环流畅度）。
2. **一闪之道**：满势一闪附带范围斩 + 处决范围扩大（强化爆发上限）。
3. **止水之道**：凝势减伤提升 + 凝势可正常移动（强化容错，适合新手）。

> 注意：因为是买断制，**苍岚的核心机制（势 / 一闪）C0 即完整**，技能树只是流派分化，不存在"必须点满才能玩"。

### 主要玩法 / 循环手法
- **短轴**：普攻 4 段（回 1 势）→ 凝势补到 2–3 层 → 看破绽放斩势。
- **长轴**：弹反 / 完美闪避攒势 + 普攻铺垫 → Boss 大破绽 → 凝满 4 层 →「破势·一闪」处决 → 终结技「奔雷」收尾。

## 垃圾箱（废弃方案存档）

> 留痕设计取舍，方便日后追溯 / 给团队看踩过哪些坑。

- **「势会随时间衰减」**——砍掉。实测心理压力太大，玩家不敢蓄、节奏变憋屈；改成"势只在释放 / 切角色时清空"。
- **「凝势完全静止不可动」**——改。初版完全站桩太死板、对玩家太惩罚；改成"可缓移 + 势越高越慢"，保留张力又给容错。
- **「满势一闪无条件秒杀小怪」**——改。太破坏数值；改成"只对破韧 / 倒地目标处决"，把爽点和"先削韧"的玩法绑定。
- **「普攻每段都回势」**——简化为"一套 4 段回 1 势"，避免无脑连段刷满势、削弱凝势博弈的意义。
