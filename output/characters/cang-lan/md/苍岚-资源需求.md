## 3. 程序需求

| 需求简述 | 实现方式 | 优先级 |
|---|---|---|
| 「势」资源（0–4 层，可视化） | 资源条 / 层数 UI；凝势积累、释放清空、普攻一套 / 完美闪避 / 弹反各回 1 势 | P0 |
| 凝势减伤 + 移速递减 | buff：凝势期间减伤 20%；移速随势层数线性递减 | P0 |
| 满势一闪处决判定 | `Skill_Charge_Release_Lv4` 命中目标若处于"破韧 / 倒地" → 触发处决分支 | P0 |
| 完美闪避取消凝势保留势 | 凝势 loop 期间完美闪避输入 → 退出凝势但不清空势层 | P1 |
| 技能树流派（3 分支） | 势之道 / 一闪之道 / 止水之道：被动 + 数值开关 | P1 |

## 4. 资源汇总 / 复用清单

| 类别 | 全新数量 | 复用数量 | 说明 |
|---|---|---|---|
| 动作 | ~16 | 通用 + 受击集（复用同体型女武者骨架） | 凝势 loop + 满势一闪是核心新增 |
| 特效 | ~8 | 普攻 / 闪避 / 弹反可族内改色复用 | 「势」蓄力视觉分级 + 满势白雷为核心新增 |
| 音效 | 拔刀 / 蓄势升调 / 一闪 / 雷鸣 各新增 | 受击通用复用 | 蓄势"升调"是听觉记忆点 |
| UI / HUD | 「势」层数条（0–4，含满势高亮） | HP 条复用通用 | 满势态要有明显"可放一闪"提示 |
