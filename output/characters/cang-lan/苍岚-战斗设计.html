      壹 · 设计思路

      苍岚的核心,是把「蓄力—释放」做成一套可被玩家主动管理的节奏资源 —— 而不是一次性的长按重击。

      她长按特殊技进入「凝势」,随时间积攒「势」层(0–4),松手按当前层数释放对应威力的斩击。层数越高越强,但凝势时移动变慢、且占用时间 —— 于是玩家永远在「再蓄一层博伤害」和「现在放保安全」之间做实时决策。满 4 层解锁「破势·一闪」居合,是她的爽点与处决手段。这套设计让一个上手不难(★★)的近战角色拥有很高的操作上限:高手用普攻 / 完美闪避 / 弹反穿插回势,在破绽窗里精准塞进一记满势一闪。

      核心机制 A · 「势」节奏管理:凝势积攒 0–4 层、按层释放 —— 不是无脑长按,而是「蓄到几层放」的实时博弈。
      核心机制 B · 金 / 紫预警契约:防御 / 应对遵守统一 telegraph —— 金 = 可格挡 / 可弹反 开进攻窗(弹反回势)、紫 = 必闪 唯一解是精确闪避(完美闪避回势 + 可取消凝势)。

      主题风格 ·「止水与奔雷」

      凝势时极静(横刀垂目、青雾流转),释放时极烈(一闪而过、雷光炸裂)。视觉靠「静—动」的强反差建立辨识度 —— 静如止水,动如奔雷。

      核心记忆点（★ 三条）

      ★ 记忆点 1
      「势」节奏管理

      不是无脑长按,而是「蓄到几层放」的实时博弈 —— 再蓄一层博伤害,还是现在放保安全。

      ★ 记忆点 2
      满势一闪 = 处决

      4 层「破势·一闪」命中破韧 / 倒地目标直接处决,高风险高回报,角色爽点。

      ★ 记忆点 3
      闪避 / 弹反回势

      防御性操作反哺进攻资源 —— 鼓励贴身博弈而非苟着蓄力。

      被动机制 ·「止水」

      • 凝势期间受到伤害减免 20% —— 鼓励「敢在敌人面前蓄力」。
      • 势每升一层,凝势时移动速度递减(4 层时几乎站桩)—— 蓄得越满、越脆、越需要走位 / 队友保护,形成张力。

      养成提升（示例项目 = 买断制技能树,无抽卡命座)

      技能树三条分支,玩家按流派点;因为是买断制,核心机制(势 / 一闪)C0 即完整,技能树只是流派分化,不存在「必须点满才能玩」。

      分支效果强化方向
      势之道普攻回势加快 / 完美闪避额外回势强化循环流畅度。
      一闪之道满势一闪附带范围斩 + 处决范围扩大强化爆发上限。
      止水之道凝势减伤提升 + 凝势可正常移动强化容错,适合新手。

      预警契约（全篇通用）

      苍岚的防御 / 应对遵守统一的 telegraph 可读性契约,贯穿图例 → 技能卡 → 应对栏 —— 同色永远同义。

      金色闪光 = 可格挡 / 可弹反 —— 弹反成功开进攻窗 + 回势
      紫色闪光 = 必闪(无法格挡) —— 唯一解是完美闪避(回势 + 可取消凝势)
      无预警 = 苍岚自身的常规招(普攻 / 凝势释放),靠节奏掌控

      设计取舍（废弃方案留痕）

      把砍掉 / 改掉的方案一并留痕,讲清「为什么苍岚最终是现在这样」(对应设计稿垃圾箱)。

      废弃 / 修改方案处置理由
      「势」会随时间衰减砍实测心理压力太大,玩家不敢蓄、节奏变憋屈;改成「势只在释放 / 切角色时清空」。
      凝势完全静止不可动改初版完全站桩太死板、惩罚太重;改成「可缓移 + 势越高越慢」,保留张力又给容错。
      满势一闪无条件秒杀小怪改太破坏数值;改成「只对破韧 / 倒地目标处决」,把爽点和「先削韧」的玩法绑定。
      普攻每段都回势改避免无脑连段刷满势、削弱凝势博弈;简化为「一套 4 段回 1 势」。

      贰 · 角色定位 + 循环手法

      苍岚是玩家角色(非 Boss),单形态全程 —— 故本章不用 Boss 的「剧情位置 / HP 阶段表 / 剧情杀」,改以队伍定位 + 推荐配队 + 核心循环(短轴 / 长轴)呈现。

      队伍定位

      项值
      定位近战输出(薙刀 · 长柄) —— 核心出伤点是满势「破势·一闪」。
      上手难度★★(中) —— 核心机制易懂,但「蓄到几层放」+ 闪避 / 弹反回势穿插拉高操作上限。
      擅长对破韧 / 倒地目标的高爆发处决;有破绽窗的单体 / 精英战。
      短板满势凝势期间移速骤降、较脆 —— 需要走位 / 队友制造安全蓄力窗。

      推荐配队（示例项目 = 单角色主导 + 可选 3 人小队）

      位置职责说明
      苍岚（主控出伤）核心 C主控输出,核心出伤点 = 满势一闪。
      坦克拉仇恨拉住敌人仇恨,给苍岚站桩凝势的安全窗。
      支援补伤 / 上易伤切人联动补伤 / 上易伤,放大满势一闪的收益。

      队友的核心职责 = 制造「能安全蓄到 4 层」的窗口;苍岚负责在窗口里把满势一闪精准砸下去。

      核心循环 · 短轴 / 长轴

      循环场景手法链路
      短轴常规清怪 / 节奏维持普攻 4 段(回 1 势)无预警 → 凝势补到 2–3 层 → 看破绽放斩势(突进斩)。
      长轴破韧 / 大破绽爆发弹反 金 · 可弹反 / 完美闪避 紫 · 必闪 攒势 + 普攻铺垫 → 目标大破绽 / 破韧 → 凝满 4 层 →「破势·一闪」处决 → 终结技「奔雷」收尾。

      循环节奏（长轴时间线示意）

      铺垫期   普攻 4 段连击(末段击退 + 回 1 势)
   防御反哺  完美闪避 / 弹反对撞 → 各回 1 势(贴身博弈,别苟着干蓄)
              └─ // 凝势 0.4s/层,可缓移、减伤 20%,势越高越慢

蓄势期   长按凝势 → 积攒至 满 4 层(此时几乎站桩,靠队友 / 走位保护)
   破绽窗    目标进入 破韧 / 倒地 / 大破绽
     └─ 「破势·一闪」居合突进 —— 穿透 + 大削韧,命中破韧/倒地直接处决 ⭐

收尾     能量满 → 终结技「奔雷」(释放期霸体 → 终刀全屏雷光冲击)

      叁 · 核心机制 + 招式一览

      📌 审阅向 — 看这一章即可掌握核心玩法:「势」蓄力机制的 4 个面 + 8 个招式分别是什么、表现如何、怎么打。预警契约:金 = 可格挡 / 可弹反 紫 = 必闪 无 = 自身常规招。具体行为 / 数值 / 几何 / 收益曲线见下一章「肆 · 落地配置细节」。

      核心机制 ·「势」蓄力(★ 四个面)

      「势」是苍岚的灵魂资源(0–4 层),围绕它有四个互相咬合的面 —— 凝势积攒、按层释放、满势处决、防御回势。这四点构成「蓄到几层放 + 防御反哺进攻」的实时博弈。

      ★ 面 1 · 凝势
      长按蓄「势」(0–4 层)

      表现横刀垂目、青雾流转,刀身随档位 1→4 由淡转浓、由青转白雷(极静)。

      玩法每 0.4s + 1 势(上限 4);凝势可缓移、减伤 20%,但势越高移速越慢(4 层近站桩)。完美闪避可取消并保留势。

      ★ 面 2 · 斩势
      按层释放(0–3 层)

      表现松手一闪而过(极烈);突进刀光长度 / 亮度随层递增。

      玩法0 层=快速横斩(保节奏) / 1–3 层=威力·范围·削韧递增的突进斩。「蓄到几层放」的决策核心。

      ★ 面 3 · 破势·一闪
      满 4 层处决

      表现居合长距突进,白雷长斩贯穿 + 命中点雷花炸裂 + 短暂定格(角色爽点)。

      玩法穿透 + 大削韧;命中破韧 / 倒地目标直接处决。高风险(满势脆)高回报。

      ★ 面 4 · 防御回势
      闪避 / 弹反反哺

      表现完美闪避残影 + 时停涟漪 紫;弹反金属火花 + 金光迸裂 金。

      玩法完美闪避 / 弹反成功各回 1 势 —— 防御性操作反哺进攻,鼓励贴身博弈而非苟着蓄力。

      招式池（skill1 ~ skill8）

      ★ = 招牌 / 核心招;标签 新增 / 复用;预警以金 / 紫色块标注(金 = 苍岚的格挡 / 弹反应对的契约色,紫 = 完美闪避应对的契约色)。

      skill1 普攻连段（4 段） 新增 无预警

      表现薙刀横扫 → 撩斩 → 转身扫 → 顿步重劈,起手轻快、末段重。

      玩法末段击退 + 回 1 势(一套 4 段回 1 势);铺垫节奏的基础链。

      ★ skill2 闪避 + 完美闪避 → 反击 新增 面 4 回势 应对紫·必闪

      表现滑步规避;完美窗 6f 触发「残影」+ 时停涟漪,可接一记上挑斩。

      玩法完美闪避回 1 势,是应对 紫色必闪 的唯一解 + 可取消凝势保留势;反击上挑接 1 浮空起手。

      ★ skill3 凝势 / 斩势（核心特殊技） 新增 面 1 / 面 2 无预警

      表现长按进入凝势(青雾流转,每 0.4s + 1 势,上限 4,可缓移、减伤 20%);松手按层释放突进斩。

      玩法0 层快速横斩(保节奏) / 1–3 层威力·范围·削韧递增的突进斩。完美闪避可取消并保留势。

      ★ skill4 破势·一闪（满势） 新增 面 3 处决 无预警

      表现4 层时释放:居合长距突进斩,白雷贯穿 + 雷花炸裂 + 短暂定格(角色爽点)。

      玩法穿透 + 大削韧;命中破韧 / 倒地目标直接处决倒地。高风险高回报。

      ★ skill5 格挡 / 弹反 新增 面 4 回势 应对金·可弹反

      表现招架姿态;完美时机弹反触发金属火花 + 金光迸裂(对齐 金色契约)。

      玩法金色预警可格挡;完美时机弹反 → 反击 + 回 1 势。贴身博弈的进攻性防御。

      ★ skill6 终结技 ·「奔雷」 新增 霸体

      表现能量满释放:高速多段斩 → 终结一刀全屏雷光冲击(动如奔雷的高光)。

      玩法大招 / AOE / 高爆发,释放期霸体;长轴收尾招。

      skill7 空中攻击（可选） 新增 无预警

      表现滞空下劈,落地一记小 AOE。

      玩法空攻 / 浮空衔接;接闪避反击的浮空起手后可衔接。

      skill8 处决（可选） 新增 面 3 无预警

      表现对倒地 / 破韧目标贴身一刀斩落 + 顿帧。

      玩法强力单段终结;满势一闪也可触发同一处决分支。

      伍 · 叙事-机制双绑定

      把人设根(性格 / 主题风格)跟机制层一一对应,每个设计点都能讲清「为什么这么做」。

      人设 / 主题根机制化实现
      "静如止水"(凝势时极静)凝势(skill3 面 1):横刀垂目、青雾流转、可缓移减伤 —— 用「极静」的演出表达蓄力的内敛。
      "动如奔雷"(释放时极烈)斩势 / 破势·一闪 / 终结技「奔雷」:一闪而过、雷光炸裂 —— 用「极烈」释放与凝势形成强反差。
      "把蓄力做成可主动管理的节奏资源"「势」0–4 层:每 0.4s + 1 势、按层释放 —— 「蓄到几层放」的实时博弈,而非一次性长按重击。
      "敢在敌人面前蓄力"被动「止水」减伤 20%:凝势期间受伤减免,鼓励贴脸蓄势;但势越高移速越慢,形成「越满越脆」的张力。
      "防御性操作反哺进攻,不许苟着蓄力"闪避 / 弹反回势(面 4):完美闪避 / 弹反成功各回 1 势 —— 把防御做成开进攻资源的主动手段,鼓励贴身博弈。
      "满势一闪是爽点也是处决"破势·一闪(满 4 层):居合穿透 + 大削韧,命中破韧 / 倒地直接处决 —— 高风险(满势脆)高回报的角色高光。
      "先削韧、再处决(不破坏数值)"处决判定绑定破韧 / 倒地:满势一闪不无条件秒杀,把爽点和「先削韧」的玩法绑定,而非纯数值碾压。
      "买断制 —— C0 即完整,流派分化而非数值墙"技能树 3 分支(势之道 / 一闪之道 / 止水之道):核心机制 C0 完整,技能树只做流派分化,不存在「必须点满才能玩」。

      柒 · 元规则命中

      本设计命中的通用元规则自检 —— 一条规则一行,讲清苍岚怎么体现它。

      元规则命中场景
      核心记忆点清晰(一句话讲清角色是谁)「蓄到几层放」的「势」节奏管理 + 满势一闪处决 + 防御回势,三条记忆点把苍岚和普通连段近战角色区分开。
      低门槛高上限(上手不难、天花板高)核心机制 C0 即完整、上手 ★★;但闪避 / 弹反穿插回势 + 「蓄到几层放」的实时博弈把操作上限拉满。
      风险-回报对称凝势越满越强,但移速骤降、越脆(减伤仅 20%)—— 满势一闪的高回报严格绑定「敢站桩蓄满」的高风险。
      进攻性防御(防御也是攒资源)完美闪避 / 弹反成功各回 1 势 + 完美闪避可取消凝势保留势 —— 防御反哺进攻,鼓励贴身博弈而非苟着干蓄。
      预警色语义全篇一致金 / 紫两色从图例 → 技能卡 → 帧表预警栏 → 应对贯穿:金 = 可格挡 / 可弹反、紫 = 必闪,同色永远同义。
      爽点与玩法绑定(不靠纯数值)满势一闪处决严格绑定「破韧 / 倒地」目标 —— 把爽点和「先削韧」的玩法绑定,而非无条件秒杀破坏数值。
      省成本优先(复用 / 单形态)通用 + 受击集复用同体型女武者骨架;角色单形态,无 Boss 的多形态 / 双场 / 入场演出额外成本。
      主题风格强反差(辨识度)「止水与奔雷」:凝势极静(青雾)vs 释放极烈(白雷)的静—动反差,从动作到特效贯穿,建立视觉辨识度。
      设计取舍留痕(垃圾箱)砍掉的「势衰减 / 完全站桩 / 无条件秒杀 / 每段回势」都在「壹」留痕 + 写明理由,可追溯。
