      壹 · 设计定位与哲学

      基本信息

      角色名称烬羽（资产名 JinYu）
      分类轴火（五行元素） · 法器（引魂灯） · 主 C（输出核心） · SSR · 上手 ★★
      使用武器法器 · 引魂灯 —— 远程播「余烬印记」+ 延迟引爆，不与人正面缠斗
      定位主 C · 输出核心；核心出伤 = 满层连环引爆收割
      上手难度★★（中） —— 上手不难（短按引燃零成本铺底），上限在「铺印分布 + 炉火节流 + 余烬窗续命 + 连环时机」四层决策
      蓝本 / 形象游走战场的引魂术者，提一盏长明引魂灯；只把印记悄悄播在敌身，待其自燃炸开。青焰长袍，灯火明灭间步步生印。

      设计哲学

      不即时打疼，而是先播种、待自燃；铺得越广，连环炸得越烈。

      烬羽的核心是延迟引爆（delayed detonation） —— 她不做即时输出，而是把伤害「先存进敌人身上、再让它自燃炸开」。普攻 / 特殊技往敌身播「余烬印记」（灼烧 DoT），印记静静累积，叠满 8 层自动引爆（命中即炸，不靠手动、不靠松手）+ 开「余烬窗」滚雪球。爽点不在「这一刀多疼」，而在把印记撒满全场、让一片敌人同一拍次第自燃。她逼玩家从「我现在打谁」切换到「我之前把印记铺到了哪、它们什么时候一起炸」—— 一种先布局、后收割的节奏。

      余烬印记（敌方 DoT 层）：普攻 / 引燃播印记，满 8 层自动引爆成火爆 AOE。
      炉火 + 余烬窗（己方资源）：攒火播印、单向不返（节流）；引爆开窗，窗内普攻强化、引爆喂引爆。

      核心记忆点

      ★ 记忆点 1
      播印记 · 自燃引爆 —— 叠满 8 层自动炸，不靠手动按键

      玩家的活是「铺得多、铺得匀」，「炸」交给敌人身上的计数器。把输出从玩家手上挪到敌身的叠层器，专注布局而非按引爆键。

      ★ 记忆点 2
      炉火经济 —— 攒火播印，单向不返，有节流

      炉火挣得到、花得掉、绝不回退；引爆不返炉火。逼玩家靠普攻 / 闪避把炉火攒回来，不能无脑长按糊脸 —— 给延迟引爆一个资源天花板。

      ★ 记忆点 3
      余烬窗 —— 引爆后强化普攻，滚雪球

      引爆开 / 刷新一扇强化窗：窗内普攻攒火更快 / 伤害更高 / 形态升级。引爆喂下一次引爆，但被「窗时长上限 + 炉火不返」双重夹住，不无限膨胀。

      主题风格

      「明灭与自燃」—— 铺印记时极克制（灯火幽幽、印记不起眼地累积），引爆时极炽烈（一片敌人同一拍连环炸开，青焰冲天）。靠「悄无声息地积累」与「轰然连环炸」的强反差建立辨识度，区别于即时爆发型主 C。

      类似玩法参考

      • DoT 流 / 引燃流的「先铺后炸」节奏感（点燃叠层 → 引爆收割）—— 参考其「延迟结算」的布局手感。
      • 「种火 → 自燃 → 连环」的延迟结算手感 —— 参考其连锁引爆的爽点。

      📝 评审备注 · 壹清空

      贰 · 核心机制

      烬羽有两套互锁的系统：敌方身上的「余烬印记」（伤害怎么发生）+ 己方的「炉火 / 余烬窗」（节奏怎么管理）。本章给机制全貌，精确数值在「肆 · 资源系统」深入；逐招表现放「叁 · 招式详解」。

      机制概览

      ★ 机制 1 · 余烬印记
      敌方 DoT 层 —— 满 8 层自燃引爆

      是什么普攻 / 短长按引燃 / 联动 / 大招命中敌人时叠「余烬印记」（每层一道灼烧 DoT，多层叠加）。单个敌人叠到 8 层（C3 后 10）即 自动引爆 —— 命中即炸，不需手动按键 / 不需松开长按。引爆 = 按层数结算的火爆 AOE + 开 / 刷新余烬窗。

      玩法玩家管「铺得多、铺得匀」，系统管「炸」。铺得越广，自燃来得越密、连环炸得越烈。

      ★ 机制 2 · 炉火
      己方资源 —— 单向不返（节流阀）

      是什么普攻命中攒炉火、余烬窗内攒更快、完美闪避 / 弹反回一小笔；长按引燃持续耗炉火（炉火 ≥ 30% 才能启动）。炉火单向不返：挣得到、花得掉、绝不回退，引爆不返炉火。

      玩法不能无脑长按糊脸，必须靠普攻 / 闪避把炉火攒回来才有下一波猛铺 —— 逼出「攒—花—再攒」的下行节奏。

      ★ 机制 3 · 余烬窗
      己方强化窗 —— 引爆喂引爆

      是什么任意引爆（满层自燃 / 大招大引爆）触发或刷新余烬窗。窗内：① 普攻攒炉火↑（×1.5）② 普攻 / 印记伤害↑ ③ 普攻形态升级（更宽扇形焰 + 播印更多）。窗时长封顶，连环引爆能刷新但不突破上限。

      玩法炸完进窗 → 窗内攒火更快、铺印更多 → 更快触发下一次自燃 → 再刷新窗。滚雪球，但被「窗时长上限 + 炉火不返」双重夹住，不无限膨胀。

      核心循环（短轴 / 长轴）

      • 短轴：普攻 4 段（播印 + 攒炉火）→ 短按「引燃」补印记 → 某敌叠满 8 层自动引爆 → 进余烬窗，窗内普攻继续滚。
      • 长轴：开局短按引燃铺底 → 炉火攒够 → 长按引燃对一片敌人猛叠印记 → 连环自燃 → 余烬窗内强化普攻继续铺 → 大招「焚天·万烬」全场印记单次连环大引爆 + 拉满炉火 + 长余烬窗，重置进下一轮。
      • 心法：永远在「再铺一层让它满 8 自燃 vs 现在炉火不够要节流」之间取舍；引爆不是终点，是喂养下一轮的起点。

      三系统互锁（节奏闭环）

      普攻/引燃 ──播──▶ 余烬印记(敌身, DoT)
   │                    │ 叠满 8 层
   │攒                  ▼
炉火(己方) ◀─不返── 自动引爆(火爆 AOE)
   │ ≥30% 启动长按       │ 开/刷新
   │                    ▼
长按引燃 ──猛叠──▶ 余烬窗(己方强化窗)
                        │ 普攻攒火↑/伤害↑/形态↑
                        └──▶ 更快铺满 → 下一次自燃(回到顶部)

      叁 · 招式详解

      招式集按《星澜纪》约定 §2：普攻连段（4–5 段）/ 闪避（含完美窗）/ 核心特殊技（机制载体）/ 终结技（能量满）/ 切人联动技（切换上场触发一次）。可选：空中攻击、弹反。每招用「表现 / 玩法」双标签。预警契约：红光必闪 / 黄光可弹反。招牌招式加 ★ 高亮 + 关联机制标签。

      Attack_01~04 普攻 4 段 ·「引魂灯 · 焰火连」 播印 / 攒炉火

      表现提灯甩出焰火弹，四段递进 —— 横甩 → 撩焰 → 转身扫 → 顿步重砸；末段灯火砸地、击退 + 留一小片火印区。

      玩法每段命中播 1 层余烬印记 + 攒炉火；末段击退 + 落一块小火印区（站区内敌人持续被叠印）。烬羽「铺底 + 攒火」的主力，不追求单段疼，追求把印记匀匀铺出去。

      Dodge / Dodge_Perfect 闪避 + 完美闪避 ·「灯影 · 错身」 回炉火

      表现踏一步灯影错身规避；完美窗内触发「青焰残影」。

      玩法完美闪避窗 6f；完美触发回一小笔炉火 + 短暂「灯影」状态（可顺势接普攻继续铺）。把防御操作反哺成攒火，鼓励贴脸博弈而非远远苟着等自燃。

      ★ Skill_Ignite（短按 / 长按） 核心特殊技 ·「引燃」 核心 播印主力 / 连环引爆触发

      表现短按挥灯洒出一束焰火，前方炸开小片火区；长按引魂灯大放，持续喷吐焰火覆盖前方扇区，灯火越烧越旺。

      玩法一键双形态 —— 短按（免费 · 开局即用）：不耗炉火，播少量印记 + 小火区，零成本铺底。长按（炉火 ≥ 30% 才启动 · 持续耗炉火）：对覆盖区内敌人猛叠印记（每 tick +层），铺得快的敌人会在长按过程中接连叠满 8 层 → 连环自动引爆。炉火耗尽 / 松手即停。

      机制要点短按是炉火见底时的零成本铺底；长按是「攒一波炉火 → 一次铺爆一片」的核心爽段。
      ★ Assist_01 切人联动技 ·「引魂入场」 联动 群播印 / 续炉火

      表现烬羽被切换上场时，提灯横扫入场，洒下一大片青焰。

      玩法切人上场触发一次 —— 入场播一片余烬印记（覆盖前方一群敌人）+ 落一块大火区 + 给自身一笔炉火。一上场就把场子「铺好底、续上火」，无缝接管后续猛铺。铺场 + 接管型联动。

      ★ Ultimate_01 终结技 ·「焚天 · 万烬」 大招 全场连环大引爆 / 资源重置

      表现引魂灯全数点亮升空，全场余烬同时被点 —— 青焰冲天，一片敌人在同一拍里连环炸开。

      玩法能量满释放 —— 对全场带印记的敌人做一次单次连环大引爆（无视 8 层阈值，把当前所有印记一次性引爆结算，按各自层数叠加伤害）+ 拉满炉火 + 拉长一段余烬窗。释放期霸体。烬羽的全场收割 + 资源重置键。

      数值要点炸完场 → 满炉火 + 长窗，立刻进入下一轮高强度铺爆。
      Parry_* 弹反（可选）·「灯身 · 格」 🟡 黄光预警 回炉火 / 补印

      表现以灯身格挡来袭，黄光时机精准 → 反推 + 一束反击焰火。

      玩法对齐黄光可弹反契约；完美时机弹反 → 对敌硬直 + 回一笔炉火 + 给该敌叠几层印记。给烬羽一个「正面接招也能转化成攒火 + 铺印」的容错手段。

      AirAttack_01 空中攻击（可选）·「落烬」 播印 / 衔接

      表现滞空甩灯洒火印，落地砸出一圈火区。

      玩法空中洒火印（对下方敌人叠印）+ 落地小 AOE 火区。给烬羽一个空中也能继续铺印的衔接，处理浮空 / 高台敌人。

      📝 评审备注 · 叁清空

      肆 · 资源系统（深入）

      烬羽有敌方印记 + 己方炉火 + 己方余烬窗三套互锁系统，贰章一句话讲不透，故拎一章深入精确数值（占位值 / 相对量，绝对面板待数值策划定）。

      系统 A · 余烬印记（敌方 DoT 层）

      载体敌方身上的叠层（每个敌人独立计数）
      每层效果一道灼烧 DoT（持续掉血），多层叠加
      叠层来源普攻每段 +1 / 短按引燃 +少量 / 长按引燃每 tick +层 / 联动入场一次性 +一片 / 弹反 +几层 / 空攻 +层
      引爆阈值8 层（C3 后 10 层）→ 自动引爆（命中即炸，非手动）
      引爆产出按「引爆时层数」结算的火爆 AOE（自身 + 周围）+ 开 / 刷新余烬窗
      引爆不产出不返炉火（节流，见系统 B）
      印记存续不随时间衰减（见垃圾箱：曾设衰减、砍掉）；引爆 / 敌人死亡时清空该敌印记
      为什么满层自动引爆而非手动：手动引爆会把节奏权又交回玩家手上、退化成「叠满→按键」的体力活，且削弱「铺得越广连环越烈」的自动连锁美感。自动引爆让玩家专注「铺」，把「炸」交给系统的计数器 —— 这是烬羽区别于普通点燃流的关键。

      系统 B · 炉火（己方资源 · 0–100%）

      范围0–100%

      陆 · 配队协力

      《星澜纪》三人队（一主双辅）+ 切人联动技 + 共享「协力能量」槽。烬羽是主 C，双辅服务她的「铺满 → 连环引爆」窗口。

      队伍模型

      位置角色职责
      主 C烬羽核心出伤 = 满层连环引爆收割（短轴自燃 + 长轴猛铺 + 大招全场大引爆）
      辅助一铺场 / 控场聚怪 + 站桩窗：把散开的敌人拢成一堆，让烬羽一次长按 / 一发联动就能把印记铺满一整片（铺得越集中，连环自燃越密）

      协同链路

      1. 铺场：辅助聚怪 → 烬羽「引魂入场」联动一发铺一片印 + 给自己续炉火。
      2. 猛铺：增益辅助续炉火 → 烬羽长按引燃对一堆敌人猛叠印 → 连环自燃。
      3. 收割：协力能量满 → 烬羽大招「焚天·万烬」全场连环大引爆 + 满炉火 + 长窗。
      4. 正反馈：烬羽每次引爆额外攒一笔协力能量 → 连环引爆 = 协力能量快速回满 → 更快轮到下一次大招 / 队友切人协力。「铺爆一片」既出伤、又喂协力，双重收益。

      协力能量正反馈（设计点）

      把「引爆」和「协力能量」绑定 —— 引爆越多、协力回得越快。这让烬羽的「延迟引爆」节奏天然驱动整队的协力节奏：她铺爆一片 → 协力满 → 队友切人放协力技 / 烬羽放大招 → 再开一轮。配队的爽点和烬羽个人的爽点同频。

      📐 命座养成曲线 · 怎么算 · 为什么 · 影响（金点 = 质变跳点 C2 / C6）

      怎么算每命座效果折算成对「循环 DPS」的增益 %，在 C0 = 100 基准上累乘成累计指数（乘区式 技能 × 增伤 × 暴击，守方乘区归一化为 1）。C1 印记 / 火区补 +6% → 106；C2 主动引燃 +15% → 122；C3 上限 +2 + 引爆系数 +6.5% → 130；C4 铺场提效 +6% → 138；C5 窗 / 炉火 +6% → 146；C6 野燔（全场连环）+40% → 205。

      为什么C2 / C6 做质变跳点（付费拐点 + 鲸鱼勋章），其余平滑补，符合 5 段付费哲学；C0 已完整可玩。

      影响205 ÷ 100 = 满命 +105%，落 +80~120% 健康带正中。质变跳点（C2 +15% / C6 +40%）明显高于平滑级（约 +6%），曲线有「两个台阶」的付费诱导感。

      ② 逐级边际（分档）

      📐 逐级边际 · 怎么算 · 为什么 · 影响

      怎么算取图①相邻命座累计指数比 − 1，得每级边际增量 %。分四档看：入门（C1 +6%）/ 质变拐点（C2 +15%，解锁主动引燃）/ 数值·铺场·循环（C3 +6.5% / C4 +6% / C5 +6%）/ 机制（C6 +40%，野燔全场连环）。

      为什么用「分档」看跳点是否落在该落的地方 —— 付费拐点（C2）和鲸鱼勋章（C6）应是边际局部峰值，平滑级应低且接近。

      影响C2（+15%）、C6（+40%）两个峰值清晰，中间档约 +6% 平稳 → 质变跳点合理，不存在「平滑级比拐点还肥」的倒挂；C6 +40% 是全图最大跳，符合「机制变质 = 鲸鱼专属」。

      ③ 增伤区稀释（余烬窗 +30% 在不同面板下的边际）

      面板已有增伤0%+60%+100%
      余烬窗 +30% 的实际边际+30%≈ +18.75%≈ +15%
      📐 增伤区稀释 · 怎么算 · 为什么 · 影响

      怎么算《星澜纪》增伤是单一加法 bucket (1 + Σ增伤)，自带增伤（余烬窗 +30%）实际边际 ≈ 30% ÷ (1 + 已有面板增伤%)。面板 0% → 30 ÷ 1.0 = +30%；面板 +60% → 30 ÷ 1.6 ≈ +18.75%；面板 +100% → 30 ÷ 2.0 = +15%。

      为什么提醒 —— 余烬窗 +30% 会被面板增伤稀释，面板越肥越不值钱。所以别按名义 +30% 估强度，也别让玩家硬堆增伤词条。

      影响烬羽走「印记叠层 / 引爆层数 / 暴击」赛道（伤害主要来自「叠多少层 × 引爆系数 × 暴击」），余烬窗 +30% 当锦上添花而非主乘区，规避加法 bucket 天花板（× ≤ ~1.5）。属性投放推荐：暴击赛道（CR:CD=1:2）+ 攻击力 + 够用充能，不主堆增伤。

      ④ 场景伤害梯度

      📐 场景伤害梯度 · 怎么算 · 为什么 · 影响

      怎么算以「群战多目标连环引爆」为 100% 基准，按可铺印记的目标数 / 是否触发连环折算其他场景：小怪 / 精英混合 ~85%（仍能铺多个）；单 Boss ~50%（只有一个计数器、无连环、靠单体反复叠满 8 层引爆）；PVP ~60%（单 / 少目标 + 印记需时间叠满，节奏被压）。

      为什么火系 AOE 主 C 的定位取舍 —— 烬羽明确是「越多目标越强」的群战收割型，单体是短板（不该单体也强，否则挤压单体 C 生态位）。

      影响单 Boss ~50% 落在 §五「群伤 C 单体 40~55%」健康带（不残废但明显是短板，符合定位）。配队 / 关卡应给烬羽多目标场景；纯单体 Boss 战她不是首选主 C。这是主动设计的取舍，不是失衡。

      引爆时层数12345678（C3）910
      引爆倍率1.0×1.4×1.8×2.2×2.6×3.0×3.4×3.8×4.2×4.6×
      📐 引爆收益 · 怎么算 · 为什么 · 影响

      怎么算引爆伤害 ≈ 基础 × (1 + 0.4 × (层数−1))，线性 —— 1 层 1.0×，每层 +0.4×；正常自燃在 8 层引爆 = 3.8×；C3 提上限到 10 层 = 4.6×（金色虚线延伸段）。

      为什么引爆收益线性随层数 —— 「铺得越满、炸得越狠」要可预期、无突变（烬羽的质变在「连环 / 全场」而非「单次倍率跳」）。C3 抬上限 = 把单次引爆天花板从 3.8× 抬到 4.6×。

      影响满层引爆 ≈ 3.8×（C3 后 4.6×）对应单次自燃的「值得」感；但烬羽真正的爆发在连环 —— 一片敌人各自 3.8× 同拍炸 + 大招全场大引爆，叠起来远超单体。引爆段相对普攻段约 8~12×（对应 §五「爆发 5~15×」）。线性曲线让玩家能心算「铺到几层炸划算」，强化「布局」决策。

      垃圾箱（废弃方案留档）

      设计过程中砍掉的方案 / 取舍留痕，强烈建议保留 —— 方便日后追溯「为什么没这么做」。

      废弃方案为什么砍 / 改成了什么
      「余烬印记会随时间衰减」砍掉。实测心理压力太大：玩家铺到一半发现前面的层数在掉、变成「跟时间赛跑」，节奏憋屈、不敢慢慢布局。改成印记不衰减、只在引爆 / 敌死时清空，把「铺」的决策权还给玩家。
      「引爆手动触发（按键引爆）」砍掉。手动引爆把节奏退化成「叠满 → 按键」的体力活，且削弱「铺得越广、连环自动炸」的连锁美感。改成满 8 层自动引爆，玩家专注铺、系统管炸。（注：C2 才以「主动引燃」形式有限度地把「提前抢炸」作为养成解锁的进阶维度放回来，而非 C0 默认。）
      「引爆返还炉火」砍掉。返炉火会让「长按 → 引爆 → 返火 → 再长按」自我循环、自燃失控成无脑刷屏。改成炉火单向不返，逼出「普攻 / 闪避攒回炉火」的下行节奏，给猛铺一波加成本、加间歇。
