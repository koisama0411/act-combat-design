      壹 · 设计哲学

      孤鸦是「镜像玩家」的决斗者 Boss —— 他也蓄「势」,整场战斗的主题是博弈:谁先蓄满、谁能在对方蓄势时打断谁。

      他不靠数值碾压,靠节奏与读招,是一个「技巧考核型」单体 Boss。叙事上他是苍岚的同门师兄、堕入修罗道的宿敌,机制上与玩家镜像对位 —— 他蓄「势」、玩家也蓄「势」,这场决斗本质是两个同源剑客的同台较量。他考核玩家三件事:① 读 金 / 紫预警 选对应对(格挡 / 必闪);② 抓 Boss 蓄势的打断窗(进攻性防御);③ P3 与 Boss 弹反对撞抢输出窗。

      核心机制 A · 拔刀蓄势(镜像「势」):Boss 收刀低身蓄势,蓄满放居合突进 —— 蓄势中被命中累计 3 下即打断、倒地大破绽。教「进攻性防御」。
      核心机制 B · 金 / 紫预警契约:威胁招一律带预警 —— 金 = 可格挡 / 可弹反 开进攻窗、紫 = 必闪 唯一解是精确闪避。

      预警契约（全篇通用）

      所有威胁招遵守统一的 telegraph 可读性契约;应对方式用通用动词描述,不锁特定角色招式。除 skill1 / skill2 这类快速普通招外,威胁招一律带金 / 紫预警。

      金色闪光 = 可格挡 / 可弹反 —— 进攻性防御开窗
      紫色闪光 = 必闪(无法格挡) —— 唯一解是精确闪避
      无预警 = 快速普通招(skill1 / skill2),靠见招拆招

      设计取舍（废弃方案留痕）

      把砍掉 / 改掉的方案一并留痕,讲清「为什么孤鸦最终是现在这样」。

      废弃 / 修改方案处置理由
      蓄势完全无法打断砍太无解、玩家只能干等;改成「命中 3 下打断」,把它变成进攻性防御的教学点。
      鸦羽·一文字 命中即秒杀改一击劝退太劝退;改成「重创非秒杀」,留容错。
      P3 全新觉醒动作集砍美术成本高;改「零新动作调色 + 提速觉醒」,既省成本又演示双场范式。
      全程无预警纯靠背招砍违反 telegraph 可读性;除 skill1 / skill2 快速普通招外,威胁招一律金 / 紫预警。

      贰 · 剧情位置 + 战斗结构

      孤鸦是苍岚的同门师兄、堕入修罗道的宿敌,作为决斗者 Boss 单体出场。战斗是单形态、多 HP 阶段结构(P1/P2/P3 + 超时狂暴) —— 不是双场 / 多形态,故下文不用 Tab,阶段以表格 + HP 条呈现。每阶段在前一阶段基础上堆叠招式(招不删、只加),并升级蓄势难度。

      HP 阶段轴

      100%70% · P1→P235% · P2→P30%
      P1 试探
      100–70%      居合试探 —— 短居合连段 + 单次拔刀蓄势(蓄满金闪 · 可打断)。设计重点:教玩家读金 / 紫、学打断蓄势。
      P2 高压
      70–35%      连段压制 —— 长连段 + 紫闪必避突进(必闪)+ 蓄势升级为 2 阶。设计重点:提速;紫闪斩逼闪避;蓄势更难断。
      P3 独眼觉醒
      35–0%      拔刀奥义 —— 觉醒态(零新动作,调色 + 提速)+ 金闪可弹反居合对撞(可弹反)+ 满蓄势奥义「鸦羽·一文字」(全屏必闪)。设计重点:终局博弈 —— 弹反对撞开窗 / 满蓄势奥义全屏紫必避。
      狂暴
      (可选 · 超时)      不停蓄势 —— 连续蓄势 + 缩短间隔。设计重点:超时惩罚,防苟战。

      双场扩展（可选思路 · 非本稿主线）

      孤鸦天然适合做双场战(同一套 Spine / 动作,靠调色 + 提速 + 台词差异做出「判若两人」的弧光,零额外动作成本):

      • 中段「初遇·孤鸦」:只放 P1–P2,他「留手」试探,HP 见底时剧情杀退场(不真死)。
      • 末段「决战·孤鸦」:P1–P3 全力,入场即觉醒态。

      本审阅稿按单场全力(P1–P3 + 狂暴)主线呈现;双场拆分为可选演进,落地时再决定是否启用 —— 故本稿没有启用「肆」的双场 Tab。

      叁 · 核心机制 + 招式一览

      📌 审阅向 — 看这一章即可掌握核心玩法:4 大记忆点机制 + 8 个招式分别是什么、表现如何、怎么打。预警契约:金 = 可格挡 / 可弹反 紫 = 必闪 无 = 快速普通招。具体行为 / 数值 / 几何 / CD / 收益曲线见下一章「肆 · 落地配置细节」。

      核心机制概览（★ 四大记忆点）

      这四个特殊设计点,是孤鸦区别于「普通连段怪」的灵魂 —— 决斗博弈感 + 进攻性防御 + 省成本觉醒。

      ★ 机制 1
      「拔刀蓄势」镜像机制

      表现Boss 收刀低身蓄「势」,刀身青焰随进度汇聚 —— 与玩家的「势」镜像对位。

      玩法玩家可在他蓄势时打断(累计命中 3 下),打断后他大破绽倒地(易伤窗)。教「进攻性防御」—— 防御不只是躲,更是抢节奏。

      ★ 机制 2
      「鸦羽·一文字」满蓄势奥义

      表现P3 全屏 紫色 巨型居合斩,屏幕泛白 + 定格,视觉高光。

      玩法必闪 + 读时机,是死亡招也是高光招;蓄力期可被打断(P3 蓄速快、难断)。改成「重创非秒杀」留容错。

      ★ 机制 3
      「独眼觉醒」零新动作觉醒态

      表现P3 不做新动作,仅调色(赤瞳 / 赤黑气场)+ 提速 15% + 蓄势变快。

      玩法同样的招、更快的节奏,逼玩家把 P1/P2 学到的读招提速复用。演示「零新动作资源条觉醒」省成本技巧。

      ★ 机制 4
      金闪居合·对撞

      表现P3 与玩家拼刀,招式释放裹 金色 闪光(可弹反契约)。

      玩法弹反成功 → 孤鸦踉跄 → 输出窗。决斗感的来源;弹反失败则接 skill6 三连惩罚。

      招式池（skill1 ~ skill8）

      应对方式用通用动词描述(不锁特定角色招式)。★ = 招牌 / 核心招;标签 新增 / 复用;预警以金 / 紫色块标注。

      skill1 居合突刺 新增 无预警

      表现拔刀直刺前冲一段,起手快、无预警的快速普通招(P1+)。

      玩法侧闪 或 格挡;距离中、CD 短。

      skill2 横扫二连 新增 无预警

      表现拔刀横斩 2 段,末段击退(P1+)。

      玩法闪 或 格;距离近、CD 短。

      ★ skill3 拔刀蓄势 新增 机制 1 蓄满金闪 · 可弹反

      表现收刀低身蓄势(镜像苍岚「势」),蓄满放居合突进 —— 释放瞬间金闪 → 可弹反(P1+,P2 升 2 阶)。

      玩法蓄势中被命中累计 3 下 → 打断,孤鸦倒地大破绽(易伤窗)。应对:打断(猛攻) 或 蓄满时 弹反 / 闪。

      skill4 鸦步后撤 新增 无预警

      表现后跳拉距,常接 skill1(P1+);意图:防玩家无脑贴身。

      玩法跟进,别被牵着走;CD 短。

      ★ skill5 紫影突进斩 新增 必闪

      表现紫闪预警 → 高速突进一斩,无法格挡(P2+);意图:惩罚玩家拉远 / 走位。

      玩法必闪(精确时机,无法格挡);距离远、CD 中。

      skill6 三连居合 新增 读节奏

      表现居合三段连击,末段浮空;中段可被打断(P2+)。

      玩法读节奏闪 / 格;距离近、CD 中。

      ★ skill7 金闪居合·对撞 新增 机制 4 可弹反

      表现金闪预警 → 居合斩,可弹反对撞;对撞成功 → 孤鸦踉跄(输出窗)(P3,决斗核心)。

      玩法弹反 或 闪;弹反失败接 skill6 三连惩罚。距离中、CD 中。

      ★ skill8 鸦羽·一文字 新增 机制 2 全屏必闪

      表现满蓄势奥义:全屏紫色巨型居合,命中重创(非秒杀)(P3;死亡招 + 高光)。

      玩法必闪(精确时机) / 蓄势中可打断;距离全屏、CD 长或 HP 触发。

      玩家应对结果易伤 / 硬直窗
