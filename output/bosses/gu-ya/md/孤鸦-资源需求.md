## 3. 程序需求

| 需求简述 | 实现方式 | 优先级 |
|---|---|---|
| 「拔刀蓄势」资源 + 打断逻辑 | 蓄势进度条（隐 / 显）；蓄力 loop 期间累计受击 3 下 → 触发 `Skill3_Break` 倒地易伤窗 | P0 |
| 金 / 紫 预警契约统一接口 | 招式可配"预警类型(金可弹/紫必闪/无)"，自动挂对应 telegraph 特效 + 判定标签 | P0 |
| 金闪居合·对撞 判定 | `Skill7` 标"可弹反"；玩家完美格挡窗内输入 → 触发 `ClashStagger` 输出窗 | P0 |
| 「独眼觉醒」零新动作觉醒态 | P3 进入时施加 buff:角色材质调色（赤瞳 / 赤黑）+ 全动作播放速率 ×1.15 + 蓄势速率提升；**不切换动作集** | P0 |
| 满蓄势奥义触发 | `Skill8` 由 HP < 15% 或周期（~20s）触发；蓄力可被打断（P3 蓄速快） | P1 |
| 狂暴（可选） | 战斗超时 → 缩短 `Skill3` 间隔、连续蓄势 | P2 |
