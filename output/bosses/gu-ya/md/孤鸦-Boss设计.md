## 战斗设计思路
孤鸦 是"**镜像玩家**"的决斗者 Boss：他也有一套「**拔刀蓄势**」，对应苍岚的「势」。整场战斗的主题是**博弈**——谁先蓄满、谁能在对方蓄势时打断谁。它考核玩家三件事:① 读 金 / 紫 预警选对应对（格挡 / 必闪）；② 抓 Boss 蓄势的**打断窗**（进攻性防御）；③ P3 与 Boss **弹反对撞**抢输出窗。

定位上他是一个"技巧考核型"单体 Boss，不靠数值碾压，靠**节奏与读招**。

## 阶段表
| 阶段 | HP | 定位 | 核心循环 | 设计重点 |
|---|---|---|---|---|
| **P1 试探** | 100–70% | 居合试探 | 短居合连段 + 单次拔刀蓄势（**可打断**） | 教玩家读金/紫、学打断蓄势 |
| **P2 高压** | 70–35% | 连段压制 | 长连段 + 紫闪必避突进 + 蓄势升级（2 段） | 提速；紫闪斩逼闪避；蓄势更难断 |
| **P3 独眼觉醒** | 35–0% | 拔刀奥义 | 觉醒态（**零新动作**，调色 + 提速）+ 金闪可弹反居合对撞 + 满蓄势奥义「鸦羽·一文字」 | 终局博弈:弹反对撞开窗 / 满蓄势奥义全屏紫必避 |
| 狂暴（可选） | 超时 | 不停蓄势 | 连续蓄势 + 缩短间隔 | 超时惩罚，防苟战 |

## 核心记忆点（★ 特殊设计点）
1. **「拔刀蓄势」镜像机制**——Boss 也蓄"势"；玩家可在他蓄势时打断，打断后他大破绽倒地（易伤窗）。教"进攻性防御"。
2. **「鸦羽·一文字」满蓄势奥义**——P3 全屏紫色巨型居合，必闪 + 读时机，是死亡招也是高光。
3. **「独眼觉醒」零新动作觉醒态**——P3 不做新动作，仅调色（赤瞳）+ 提速 15% + 蓄势变快。演示 [dual-arena-boss.md](../../../../references/dual-arena-boss.md) 的"零新动作资源条觉醒"省成本技巧。
4. **金闪居合·对撞**——P3 与玩家拼刀，弹反成功 → 孤鸦踉跄 → 输出窗。决斗感的来源。

## 技能模组（按阶段堆叠）
- **P1** = 居合突刺 `skill1` + 横扫二连 `skill2` + 拔刀蓄势·一阶 `skill3` + 鸦步后撤 `skill4`
- **P2** = P1 + 紫影突进斩 `skill5` + 三连居合 `skill6` +（`skill3` 升级为 2 阶蓄势）
- **P3** = P2 + 觉醒态 + 金闪居合·对撞 `skill7` + 满蓄势奥义「鸦羽·一文字」`skill8`

## AI 策略
| 阶段 | 策略 |
|---|---|
| P1 | 中距居合试探；每 ~8s 一次 `skill3` 蓄势（给玩家练打断）；玩家拉远 → `skill4`/`skill1` 拉近 |
| P2 | 连段压制，`skill3` 升 2 阶且更频繁；玩家拉远 → `skill5` 紫闪突进惩罚；近身 → `skill6` 三连 |
| P3 | 入场即「独眼觉醒」（调色 + 提速）；以 `skill7` 金闪对撞拼刀为主；HP < 15% 或每 ~20s → `skill8` 奥义 |
| 狂暴 | 超时进入，`skill3` 蓄势不停、间隔缩短，逼玩家速战 |

## 连招策略
- **P1**:`skill1 →`（近）`skill2` /（远）`skill4 → skill1`；周期插 `skill3`（蓄势，权重 20%）
- **P2**:`skill6 →`（玩家远）`skill5` /（近）`skill2`；`skill3`（2 阶蓄势，权重 30%）
- **P3**:`skill7`（对撞）→ 弹反失败则 `skill6`；蓄满或 HP 触发 → `skill8` 奥义

## 双场扩展（可选）
孤鸦天然适合做 [dual-arena-boss.md](../../../../references/dual-arena-boss.md) 的**双场战**:
- **中段「初遇·孤鸦」**:只放 P1–P2，他"留手"试探，HP 见底时剧情杀退场（不真死）。
- **末段「决战·孤鸦」**:P1–P3 全力，入场即觉醒态。
- 两场**同一套 Spine / 动作**，靠调色 + 提速 + 台词差异做出"判若两人"的弧光，零额外动作成本。

## 垃圾箱（废弃方案）
- **「蓄势完全无法打断」**——砍。太无解、玩家只能干等；改成"命中 3 下打断"，把它变成进攻性防御的教学点。
- **「鸦羽·一文字 命中即秒杀」**——改。一击劝退太劝退；改成"重创非秒杀"，留容错。
- **「P3 全新觉醒动作集」**——砍（美术成本高）；改"零新动作调色 + 提速觉醒"，既省成本又演示 dual-arena 范式。
- **「全程无预警纯靠背招」**——砍。违反 telegraph 可读性；除 skill1/2 这类快速普通招外，威胁招一律金 / 紫 预警。
