# 角色原型扩展(配套 character-archetypes)

本文件存放 [角色原型库](character-archetypes.md) 的扩展原型(F-Q 型,共 12 种)详细描述。
主文件保留核心 5 种(A-E)详细;F-Q 业界调研新增的详细在本文件。

> 说明:原型头部不再绑定固定的「角色分类轴」标注——各项目的角色分类轴(职业 / 定位 / 稀有度 / 武器型……)由项目自定义,见 [角色原型库](character-archetypes.md) 开头说明。本文件每个原型只描述其**机制套路 + 业界样本**,可自由映射到你项目的分类体系。

### 原型 F:多态共享资源池(黑神话棍法 / Yakuza 4 流派)🆕

**核心套路**:
1. **3-4 个攻击形态共享同一资源条**,但每态有独立招牌切手技
2. **战斗中段无缝切态**(像 Dante 风格切换,不打断 combo)
3. 每态独立天赋树(玩家选 build 时决定哪态强化)
4. **不同形态克制不同敌人**——逼玩家根据战场切态

**样本**:
- **黑神话棍法三态**(劈/立/戳),共享棍势能量:
  - 劈棍:破棍式(出招瞬间化解攻击,无敌帧)
  - 立棍:风云转(蓄力攀棍头避开地面所有攻击)
  - 戳棍:退寸进尺(后撤+前冲攒刺)
- **Yakuza 桐生四流派**(粗暴/匹敌/疾风/堂之龙):
  - 粗暴:默认 4 连普攻,可抓可摔可拾物
  - 匹敌:李小龙体术,6 连快短拳,机动性最强
  - 疾风:重型流派,3 连重攻击,可单手举摩托车
  - 堂之龙:Kiryu 真传,后期集前三所长

**适用方向**:高玩向,讲究"形态-时机-资源"三层管理。**适合做"长情角色"——同一角色玩到多套体验**

**对运营向(抽卡/养成)产品的启示**:**新流派比新角色便宜**——可以以"形态切换包"作为进化路径

**资产关键**:同一资源条 + N 套独立动作 + 无 CD 切换瞬间动画

### 原型 G:形态身份切换(FF16 Eikon)🆕

**核心套路**:
1. **8 套形态身份(Eikon),最多同时装 3 个,L2 实时切**
2. 每个身份带 4 个独家技能 + 1 个 "Core Ability"(长按蓄力身份招表达定位)
3. 30 秒内能换 3 次身份(单角色像 8 个职业)
4. 跨形态技能可继承(把 A 形态的招带到 B 形态槽)

**样本**:
- FF16 Eikon 8 套:Phoenix(机动) / Garuda(削韧) / Titan(反击) / Bahamut(远程) / Odin(标记普攻) / Ramuh(陷阱) / Shiva(控场)

**适用方向**:旗舰级角色,玩家"一个角色玩很久",高养成上限

**与原型 E 双形态的区别**:Eikon 是 N 形态自由配 3,不是固定双形态;更类似"装备槽位"思路

### 原型 H:Pressure 工程师(Tifa 模式)🆕

**核心套路**:
1. 角色自身伤害不高,但**专门拉 Stagger 倍率**
2. 普攻命中给敌人 Pressure 状态;命中 Pressure 中敌人则升级到 Stagger
3. Stagger 期间全队倍率从 160% → 230%+(Tifa 独家 +70% 工程师价值)
4. 自身有"气劲槽"1→2→3 三段,每段普攻链更长伤害更高

**样本**:
- FF7 Rebirth Tifa(Unbridled Strength 气劲槽)
- 绝区零安比(失衡积累效率最高)

**适用方向**:破韧 / 击破型角色的"乘数化升级"——不只是破韧人,而是"放大队友输出窗口的工程师"

**关键设计**:Tifa 独立看伤害不强,但配队后**全队 DPS 提升明显**——衡量不是单体伤害而是"团队带飞效率"

### 原型 I:召唤指挥(鬼泣 V / FF7 Cait Sith)🆕

**核心套路**:
1. **玩家本体输出弱**,主要靠召唤兽/宠物输出
2. 召唤兽有独立血条/HP,死了要重召
3. 玩家本体负责"补刀 finisher"——召唤兽打死敌人需要本体补一刀
4. 召唤兽可暂时被玩家直接操控(Demon Slave 范式)

**样本**:
- 鬼泣 5 V(召唤 3 只魔兽,自己只补刀)
- FF7 Rebirth Cait Sith(召唤莫古利后切骑乘巨锤)
- 哈迪斯 2 法杖(远程攻击中靠召唤增伤)
- 操虫棍(派遣 Kinsect 吸怪精华)

**适用方向**:策略型副输出 / 增益型,适合"棋盘式 RTS"思维玩家

**关键设计**:召唤兽不能太强(玩家失去主导感),也不能太弱(本体输出又不行)——平衡点在"召唤兽提供决策维度而非伤害爆发"

### 原型 J:武器组合槽(猎天使手脚 4 槽 / Aspect 系统)🆕

**核心套路**:
1. 角色有 4 个武器槽位(2 手 2 脚 / 主副 + 备主备副),可挂载不同武器
2. 槽位组合改变 combo 招式集(不只是切武器,是混合武器节奏)
3. 还可挂第二套武器组合,combo 中切换不掉招
4. Bonus:每把武器的隐藏 Aspect 完全重做招式逻辑(同武器三套打法)

**样本**:
- 猎天使魔女(手 + 脚 4 件武器,Punch + Kick 分离按键)
- 哈迪斯 Aspect 系统(同一武器不同 Aspect 完全重做)
- 战神武器+卢恩(武器固定但卢恩技能可挂)

**适用方向**:深度操作向,鼓励"编织"combo

**关键设计**:**手脚分离按键**——拳为 Y / 腿为 B,玩家可在 combo 中编织切换。这是猎天使 combo 编辑系统的灵魂

### 原型 K:博弈规则修改型(永劫角色技能 / DD2 Trickster)🆕

**核心套路**:
1. 角色技能**不是放伤害,是修改战斗规则**
2. 例:冻结技 = 让敌人下一次蓄力被无视;时间技 = 让我方弹反窗口变大
3. 给"博弈层"加新维度,而不是数值放大
4. 短 CD 高频使用,塑造"节奏控制者"形象

**样本(基础版)**:
- 永劫无间顾清寒(冰冻 + 把追击对手原地停车)
- 永劫无间天海(抓投绕过振刀)
- 哈迪斯 Boon 染色(改变招式底层逻辑)

**极致变体:Trickster 型(完全不打输出)🆕**
- **DD2 Trickster**(伪装师 / 龙之信条 2):**独家武器香炉(Censer)** + **零直伤** + 靠**制造幻象诱使敌人自相残杀**
- 玩家完全不输出,但通过"幻象 + 操控 + 控场"赢得战斗
- 是"完全不打 DPS"的极致支援型设计——大胆突破"主 C+辅助" 二元

**适用方向**:辅助 / 控场 / 战术指挥型,适合"想赢得脑子",非"想赢得手速"

**关键设计**:技能效果是"博弈规则修改",**不是数值大小**——玩家用对了能扭转战局,用错了无效。Trickster 变体进一步打破"输出 = 价值"的运营向(抽卡)通病

### 原型 L:远程姿态独立型(MH 弓弩 / Jedi Survivor Blaster)🆕

**核心套路**:
1. **整套招式集都是远程**,没有近战分支
2. 蓄力 / 弹种切换 / 装填 / 距离判定 等"远程专属操作"
3. 走位 + 站桩输出节奏(不靠贴脸 combo)
4. 适合"狙击手 / 法师 / 弓兵"角色形象

**样本**:
- MH 弓(蓄力多重箭)
- MH 轻弩(速射+元素特化)
- MH 重弩(慢速大伤+贯通弹)
- Jedi Survivor Blaster 姿态(光剑+爆能枪)

**与"近中远兼修控距型"的区别**:那类是"近中远兼修+控距",原型 L 是"纯远程,无近战"——更纯粹的"狙击位"(参考样本:潜行者 / 狙击手类角色常做成"近中远三段切换",而本原型刻意舍弃近战段)

**关键设计**:远程角色容易"无脑站桩",需要加"装填管理 / 弹种切换 / 站位预判"让玩家忙起来

### 原型 M:变形武器双形态(Bloodborne Trick Weapon)🆕

**核心套路**:
1. **同一武器有 2 个形态,L1 一键切换,动作完全不同**(等于"一武器两武器类")
2. **变形即攻击(Transform Attack)**——按 R1+L1 同时切换 = 变形动画自带攻击,无后摇衔接
3. 形态间动作互补:折叠态(快 + 灵活)/ 展开态(慢 + 高 stagger)
4. 适合"动作流"高玩

**样本**:
- **BB Saw Cleaver**:折叠 = 短刀(快速 r1 + 灵活上挑)/ 展开 = 大锯(慢但高 stagger,对兽人特攻)
- **BB Ludwig's Holy Blade**:直剑态(平衡)/ 双手大剑态(超大范围 + 高伤) —— 同一武器,两个不同武器类
- **BB Chikage 千景**:普通刀 / 染血态(消耗 HP 换血属性伤害,"危险高回报")
- **BB Whirligig Saw**:锤态(慢但破势)/ 旋转锯(连击对小怪秒杀)

**与原型 F 多态共享资源池的区别**:
- 原型 F(黑神话棍法 / Yakuza 4 流派):3-4 态 / 共享资源池 / 每态独立天赋树
- **原型 M(BB Trick)**:**2 态 / "变形即攻击"无后摇衔接** / 形态切换是"招式"而非"模式选择"

**关键设计**:**L1 切换不只是"换武器",而是"招式" —— 切换瞬间能打人**。这是 ACT 武器系统的"动作化"——按 L1 不再是"打开菜单",而是"出招"。

### 原型 O:武器组装型(Lies of P 刃+柄)🆕

**核心套路**:
1. **武器拆成"刃 + 柄"两部分,可自由组合**(40 × 40 = 1600+ 组合)
2. 刃决定**伤害类型 + 普攻动作**
3. 柄决定**技能(Fable Art)+ 蓄力机制 + 数值面板**
4. 同一角色,组装出几十种"build 起点"

**样本**:
- **Lies of P**:刃 + 柄 / 1600+ 组合 / 每件武器 2 个 Fable Art
- **战神 Ragnarök 武器 + 卢恩**:类似但更简单(2 槽位)
- **法环 武器 + 战灰 + 词条**:三层堆叠

**与原型 J(武器组合槽 4 件套)的区别**:原型 J 是"挂多件武器同时用",**原型 O 是"组装一件武器但模块化"**——玩家做"build 思考"。

**关键设计**:让"换零件"成为养成系统的一部分。

### 原型 P:工具箱辅助槽(Halo Equipment / Legion Arm)🆕

**核心套路**:
1. **1 个独立"配置位"**,可从 4-8 选 1-N 件辅助技能
2. **热切换 + 共享 CD**(4-6 秒),不能"无脑全堆"
3. 配置位**独立于"职业固有招盘"**
4. 每 4 秒决策"我现在需要冲刺/防御/透视/钩拽"

**样本**:
- **Halo Infinite Equipment**:Grappleshot / Drop Wall / Threat Sensor / Thruster / Repulsor
- **Lies of P Legion Arm**:8 种机械义肢
- **Hi-Fi Rush 队友召唤**:Peppermint / Macaron / Korsica / CNMN

**与原型 N(工具箱猎人型)的区别**:原型 N 是"5+ 工具常驻按敌人切",**原型 P 是"从池子选 1-N 上场,战前 build 决策"**。

**关键设计**:**Grappleshot 思路**——把"近战格斗"和"远程射击"距离瞬间消除。可作为偏机动的副输出招牌。

### 原型 Q:后台战斗型(绝区零 Trigger / 崩铁记忆 Memosprite)🆕

**核心套路**:
1. **角色不切人到场上也能持续输出 / 加 buff / 削韧**
2. 后台触发条件:前台普攻 / 受击 / 特定事件
3. 或:**召唤独立战场单位**(Memosprite / Spirit Ash)有独立 HP / SPD / 受控状态
4. 解决"必须切人才有用"的强制切换体验

**样本**:
- **绝区零 Trigger**:**首个"后台击破"** — 不切人也能加 Daze
- **崩铁记忆命途**(Aglaea 缝纫女 / 遐蝶 / 风堇):独立行动单位
- **原神行秋/夜兰**:Off-field 持续输出
- **MH 操虫棍 Kinsect**:派遣虫吸怪精华

**关键设计**:**让"配队中的副输出永远在工作"**——把"切人爽感"升级为"全队并行作战"。是 2024-2026 运营向(抽卡)产品"召唤物作为独立单位"趋势的代表。

### 原型 N:工具箱猎人型(Witcher 5 印记 + 油 + 药水)🆕

**核心套路**:
1. **不靠"哪招最强",靠"哪招对当下问题最适合"**
2. **5+ 种工具技能**(各有专属用法,无最强解):
   - 推(物理冲击/击倒)
   - 火(燃烧 DOT / 烧甲)
   - 陷阱(地面减速 / 强制鬼魂实体化)
   - 护盾(1 次格挡 + 反伤)
   - 控制(魅惑敌人变盟友)
3. **战前准备 = 战斗一半**:读怪物图鉴 → 涂油(对特定怪物 +X% 伤害)→ 喝药水(临时 buff,有"毒性槽"管理)→ 选印记
4. 适合"研究型玩家",战斗节奏慢但深度高

**样本**:
- **Witcher 3 Geralt** + 5 印记(Aard / Igni / Yrden / Quen / Axii)+ 油 + 药水 + 炸弹
- **怪物猎人**(读图鉴 + 调装 + 选弱点弹) —— 同源思路
- **黑神话 4 法术**(定身 / 铜头铁臂 / 聚形散气 / 身外身) —— 部分类似

**关键设计**:**"准备 = 70% 战斗" 哲学**——把"提前喝药/涂油/选印记"做成战斗一部分,而非战前菜单。Witcher 3 让玩家**"在 boss 战前研究 30 分钟再开打"**,这种"战前准备"是辅助位 / 协同位可以学习的——把准备搬进战斗的前 3-5 秒。

**与原型 K(博弈规则修改型)的区别**:
- 原型 K:技能改变战斗规则(冰冻 = 让敌人下次蓄力被无视)
- **原型 N**:技能是**工具箱里的特定钥匙**,每个工具有"专属对应问题"(锐爪鬼魂必须用 Yrden,水鬼必须用 Quen)

## 候选未沉淀原型(来自格斗游戏调研)

格斗游戏 35 年沉淀的"角色独家机制"金矿,以下 10 种候选,实操后可纳入主原型库:

| 候选 | 代表 | 核心 |
|---|---|---|
| 资源条积攒型 | SF6 Manon Medal(投技攒勋章升级)| 攒"印记/层数"换爆发,跨回合保留 |
| 自损爆发型 | GG Strive Nagoriyuki Blood Gauge | 主动扣血进 Blood Rage,刀型攻击范围/伤害三连升 |
| 武器投放型 | BlazBlue Litchi / Strive Ramlethal | 把武器扔出去,本体招表临时变更,需回收 |
| buddy 召唤型 | BlazBlue Zato-1 Eddie / Strive Asuka | 招式由 buddy 释放,本体脆,puppet 操作 |
| 形态切换型(姿态多套) | VF Lei-Fei 五形 / DOA Brad Wong / Vanessa 双姿态 | 一角色实际多套招表 |
| 反击专精型 | VF Aoi / BBCF Hakumen / DOA Helena | 等对手出招才有强招 |
| 陷阱标记型 | BlazBlue Mu-12 Stein / SF6 JP Stigma | 屏幕放点,后续触发联动 |
| install 型(暂时超人) | Skullgirls Eliza / Strive Sol Dragon Install | 满血/满气一次性进入超人形态 |
| 猜方向反击型 | DOA 三角系统 Hold | 反击需要选方向/类型 |
| 招式借用型(模仿者) | DOA Honoka(模仿其他角色招) | 学习 boss 招式后转化为玩家可用招 |

**最值得学的 2 个范式**:
- **BlazBlue D 键哲学** = 同一按键、独家功能(降学习成本+提辨识度)
- **Skullgirls 叙事+机制双绑定** = 飞行不是"她会飞",是"她背后有寄生虫 Buer Drive"
