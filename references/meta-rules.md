# 元规则 + 机制创新法则(核心思维内核)

本文件是这套 ACT 战斗设计方法论的**思维核心**——让 AI 策划能"举一反三"。
- **第一部分:46 条元规则**(设计判断准绳)
- **第二部分:机制创新法则**(从灵感种子推导新机制的思维链)

---

## 一、46 条元规则

### A. 角色差异化(规则 1-3)

#### 1. 差异化在"操作动词"层,不在数值层

每个角色应该有**独占的按键玩法**,不只是数值差异。

**样本**:鬼泣 5 三主角各自玩法不同 / MH 14 武器异化 / 鸣潮共鸣回路。

**反例**:同职业的高稀有度角色和低稀有度角色,区别只是攻击力数值。

---

#### 2. 资源条 = 玩家机制识别 UI

资源条让玩家"读懂"角色,不只是数值容器。**形状/颜色/动效要异化**——散华是花瓣形,安可是火焰。

**样本**:黑神话棍势 / Tifa 气劲三段 / 绝区零失衡条 / 操虫棍三色 essence。

---

#### 3. 临时形态切换 = 单角色多职业的银弹

一个角色能玩出 N 种职业体验,是长情角色的关键。**形态切换不只是数值变化,整套招式集换**。

**样本**:FF16 Eikon 30 秒一切换 / 对马 4 姿态 / 黑神话棍法三态 / Yakuza 桐生 4 流派。

---

### B. 决策与化学反应(规则 4-7)

#### 4. 化学反应 = 决策深度上限

"组合"比"单角色"产生更多决策深度。多角色切换时**触发额外效果**,把组队从"叠 buff"升级为"互相催化"。

**样本**:原神元素反应 / 异环异能环合 8 种 / 哈迪斯 Boon 染色。

---

#### 5. Boss 反向设计:每招/每 Boss 强迫学一个机制

不是先设计 boss 再想招,而是**先想"要教玩家什么"再设计**。**每个 boss 要回答:"玩家学到本游戏哪个核心动作?"**

**样本**:只狼苇名一心四阶段(四种武器流派教科书)/ 战神 Gna(综合考试)/ MGR 参议员 Armstrong。

---

#### 6. 防反系统颜色语义化(不只红光)

视觉信号要细分,**至少 3 色,最好 4 色**——红/黄/蓝/紫各对应一类玩家反应。视觉语言标准化是 ACT 出圈的硬指标。

**样本**:只狼危技刺/横/抓 / Jedi Survivor 红黄蓝白 / 剑星红黄蓝紫。

---

#### 7. 团队联动技升级:双角色并行 / 多段联击 / 三层 burst

团队联动技(切人技 / 协同技)不是"瞬时大招演出",而是"团队连击系统"。**让玩家有可操作空间**——按一下进入新玩法 5-10 秒,不是按一下放动画。

**样本**:猎天使 Demon Slave(协力召唤角色可玩家手动操控)/ FF7 Rebirth Synergy 两层 / Relink SBA → Chain → Full Burst。

---

### C. 平衡与避坑(规则 8)

#### 8. 设计警示:单一机制收益过高 → 其他玩法贬值

任何机制的收益不能让其他玩法变成"不值得用"。

**样本**:卧龙化解被诟病(收益过高) / FF16 Limit Break + Stagger 滚雪球(深度不足批评)。

---

### D. 击破与节奏(规则 9-12)

#### 9. 架势条 / Stance Break 显式化 = 击破定位的载体

HP 之外的**第二条胜负条件**。**所有 Boss 加显式韧性条** + 击破职业专门高效削韧 + 削破后给 2-3s 暴击窗口 + 高伤倍率(类 FF7 Rebirth 160→300%)。

**样本**:只狼架势条 / 艾尔登 Poise / 绝区零失衡条 / MH wound + Focus Strike。

---

#### 10. "Pressure → Stagger → 倍率窗口"三段式 > 纯 HP 阶段化

阶段切换的触发条件是**玩家行为**,不是血条阈值。

**样本**:FF7 Rebirth Pressure → Stagger 160-300% / FF16 Stagger Will / MH Focus Mode + Wound。

---

#### 11. "打过 Boss 即学到招"(收编爽感)= 成长闭环

Boss 招式不只是给玩家克制,还是后续可用资源。

**样本**:黑神话精魄变身 / 对马传说技 / MH 解锁召唤兽 / 火焰纹章风花雪月教化。

---

#### 12. 三方制衡博弈 > 二元对立

石头剪刀布比"是/否"判定更深。

**样本**:永劫无间(连击/蓄力霸体/振刀,纳什均衡)/ 只狼(刺/横/抓)/ 对马 4 姿态。

---

### E. Boss 设计与生态(规则 13-16)

#### 13. AI 生态化:Boss 像生命体不像状态机

Boss 应该"对玩家行为做出反应"——不躲多次 → enraged;打多次同部位 → 部位"流血→折断→不能再用该招";30 秒无攻击 → exhaust。

**样本**:MH 怪物 AI(愤怒/疲劳/瘸腿/Turf War)/ 对马恐惧扩散 / 红色沙漠 Reed Devil 距离行为树。

---

#### 14. Boss 出圈 = 美学 + 独家招式 + 玩家外号

机制独家性 > 数值堆砌。**设计 Boss 必问:"玩家会给哪招取外号?"**——预测不出来就不够独特。

**样本**:玛莲妮亚水鸟乱舞 / 苇名一心雷返 / Magnamalo 怨虎龙 / Sephiroth Reborn / 猩猩假处决反转。

---

#### 15. 完美防御 + 反击形式多样化

同一闪避动作,**接不同按键给不同反击效果**——普攻接=突进近战 / 特殊技接=远程 / 大招接=AOE。

**样本**:FF16 Precision Dodge(二选一反击)/ 猎天使 Witch Time / MGR Zandatsu / 对马 Standoff。

---

#### 16. "扔武器 / 伪 KO 反转"= 同 HP 条两个 Boss

单 boss 单 HP 但中段切换整套招式集——**中段视觉强提示 + 招式表换掉,玩家心理预期被打破**。

**样本**:艾尔登戈弗雷→Hoarah Loux / 只狼猩猩 / Sephiroth Reborn / 黑神话黄风大圣。

---

### F. 节奏与跨界(规则 17-19)

#### 17. 闪避奖励要给"战略价值",而非只是 i-frame

完美闪避不只给"短暂无敌",更要给玩家"决策资源"——免费一次特殊操作 / 重置机制 / 强化窗口。

**样本**:战双超算空间(闪避 + **免费一次三消**)/ 崩 3 时空断裂 + 重置 CD / Hades 2 Ω 蓄力变招。

**🆕 17b. 衰减触发原则(反制打断的正反馈)**:玩家成功**反制/打断 Boss 的蓄力招或大招**时,Boss 的威胁应**衰减**,而不是"原威力提前触发"。

- ❌ 反例:玩家打断 Boss 蓄力 → Boss"把蓄到一半的招原威力提前放出来" → 威胁没变,玩家反制了个寂寞
- ✅ 正解:玩家打断 → Boss **失控,威胁衰减**(招式哑火 / 缩水 / 打偏 / 自伤) → 反制有清晰正反馈
- 衰减后可让 Boss **自损 / 露破绽 / 进 Stagger**,把"我打断了它"变成可见的奖励
- **样本**:怪物猎人 系列打断怪物蓄力后其踉跄露背;只狼 识破突刺后敌人失衡——反制都带可见的状态回报,而非"威胁照旧"

---

#### 18. Build / 装备应"改变手感",不只是数值

**应该改变动作判定 / 招式偏向 / 资源管理方式**——让"换装备"= 换玩法流派。

**样本**:AC6 机甲组装(腿型决定运动)/ 法环战灰系统 / NieR 芯片系统(HP/UI 可拆) / 哈迪斯 Aspect(同武器不同 Aspect 完全重做)。

---

#### 19. 大胆做"系统跨界",避免单一玩法疲劳

ACT 项目可在主战斗外加入回合制 / 解谜 / 卡牌战 / 节奏小游戏 —— 单一系统疲劳是二游运营最大敌人。

**样本**:Like a Dragon 7-8(ACT → 回合制)/ NieR 9S Hack 段 / 哈迪斯"剧情 + roguelite 融合"。

---

### G. 资源与节奏(规则 20-23)

#### 20. 血/资源同槽双用 = 最纯粹的 souls-like 经济

血量和大招资源同根经济,玩家被迫"自选回血 vs 输出"。**双向资源**比单纯 CD 经济深 10 倍。

**样本**:Hollow Knight SOUL 槽(回血 vs 烧法术) / Silksong Silk(必须满槽才能用) / Bloodborne Rally(被打 → 0.5-1s 内攻击回血)。

---

#### 21. 形态切换 = 招式池切换(不是数值调整)

形态切换不是"难度调整",而是"换游戏玩"——每形态完全换招式池、模型、美术。

**样本**:Cuphead 每 boss 3-4 形态 / Bloodborne Trick Weapon 一键切换 + 变形即攻击 / FF16 Eikon。

---

#### 22. Boss 叙事阶段切换(觉醒/变身播片)

阶段切换不只"加招式池",而是"播一段角色弧光"——玩家心理预期被改写。**强视觉提示 + 叙事意义 + 招式美学转换**。

**样本**:Bloodborne Ludwig(兽变人,自报"Holy Blade") / Father Gascoigne 人/兽对照 / Lady Maria 武器染色三阶段(银 → 红 → 焰)。

---

#### 23. 战前准备 = 战斗的 70%

不只是"开战后的操作",而是"准备 → 战斗 → 复盘"循环。把战前准备搬进战斗节奏。

**样本**:Witcher 3 涂剑+喝药水+选印记 / Hollow Knight Charm 装备槽 / MH 调装。

---

### H. 高阶设计(规则 24-26)

#### 24. Snapshot vs Dynamic 是 Buff 设计的隐藏发动机

Buff 触发瞬间"快照"属性 vs "实时随当前角色变化"——决定能否堆叠多个 buffer。**每个 buff 必须明确是 snapshot 还是 dynamic**。

**样本**:原神国家队(行秋/香菱 snapshot 班尼特 buff 整 10 秒持续) / FF7 Rebirth Synergy。

---

#### 25. "脱手"是 3D ACT 配队的灵魂

班尼特/行秋/夜兰是脱手强角色——**只用 1 个角色操作但 4 个角色都在输出**。**ACT 团队联动 / 切人机制应有"脱手"机制**——切人后被切下的角色留 5-10 秒持续输出/buff/循环。

**样本**:原神行秋 E+Q / 香菱旋火轮 10 秒 / 绝区零 Anomaly 角色 / 战双 sub-DPS QTE。

---

#### 26. 死亡惩罚革新 = 把"重置进度"转化为"持续可玩难度曲线"

死亡不只是"重置",而是给玩家"成长 token / 难度调整 token"。**"失败是练功不是惩罚"**。

**样本**:Sifu 年龄机制(死 = 老 = 难) / Hades 死亡 = 叙事推进 / Dead Cells 死亡解锁未来 build。

---

### I. 业界顶级范式(规则 27-32)

#### 27. 统一资源管理槽:"一管走天下"

一条资源同时驱动攻防多种机制 → 每个微决策都伴随机会成本。

**样本**:街霸 6 Drive Gauge(5 段统一资源驱动 5 种机制) / AC6 EN 能量 / 黑神话棍势。

---

#### 28. 节拍同步攻击(Just Timing)

招式时机软同步 BGM 节拍 → 踩拍触发额外奖励 → **不强制但鼓励**(避免劝退节奏不准的玩家)。

**样本**:Hi-Fi Rush(踩拍 = +20% 伤害) / 剑星 Tachy / Crypt of the NecroDancer。

---

#### 29. 决策密度节拍(30 秒法则)

每 30 秒 / 10 秒一次有意义战斗决策。**单纯按 ABCD 循环无决策 = 战斗设计失败**。

**样本**:Halo 30 秒法则(Infinite + Equipment 后变 10 秒) / 永劫无间三方博弈 / 只狼每秒决策。

---

#### 30. 同 Boss 多变体 = 防 powercreep 的"克制矩阵"

同一 Boss 用多种"装甲/属性/弱点变体"出现,强制玩家备多套阵容。**N×N 克制表是防 powercreep 的天然约束**。

**样本**:苍蓝档案 Grand Assault(同 boss 3 装甲) / 崩铁三大端游模式(MoC/PF/AS) / MH 不同种+亚种+古龙。

---

#### 31. 单角色机制变量 ≤ 2 个核心锚点

单角色"招牌机制"控制在 1-2 个变量——超过会让玩家"看了攻略才会用"。

**样本**:
- 正例 原神胡桃(血祭 + 重击 = 2 个变量)
- 反例 鸣潮 Phoebe(Confession + Frazzle + 协奏 + 重击 → 社区批评"理解成本 > 强度收益")

**设计要点**:第 3+ 个机制只能是"招牌的派生",不是独立变量。

---

#### 32. 玩家反馈响应速度 = 长线运营核心

2024-2026 二游赛道:**"对玩家反馈的响应速度"决定口碑生死**。

**样本**:鸣潮一年逆袭 / 绝区零 2.0 "重新开服" / 米哈游 2024 流水下滑 23%(教训)。

**设计要点**:数值/技能效果可热更新;每版本必有"老角色加强";玩家反馈渠道直通设计师。

---

### J. 格斗游戏沉淀(规则 33-42,来自 35 年格斗游戏哲学)

#### 33. "机制即定位":独家机制和角色定位强绑定

不靠数值差异,靠"独家资源/独家招"绑死定位——grappler 攒投技勋章(SF6 Manon),zoner 攒陷阱标记(JP Stigma),shoto 不攒条玩通用系统。

**样本**:SF6 全角色头顶独家图标 / BBCF D 键 = 角色 DNA。

---

#### 34. 叙事 + 机制双绑定

飞行能力要有叙事根源——不是"她会飞",而是"她背后有寄生虫 Buer Drive"。

**样本**:Skullgirls Painwheel(飞行靠寄生虫) / Cerebella(力量来自帽子) / KOF Leona Riot of Blood(暴血状态来自血统)。

---

#### 35. "统一接口 + 独家功能":降学习成本提辨识度

同一按键、不同角色按下后效果天差地别。**输入统一,效果独家**——降低操作复杂度,提升角色差异。

**样本**:BlazBlue D 键 30+ 角色 30+ 套独家功能。

---

#### 36. "层数 → 质变跳跃":资源不是数值线性 +

资源攒到不同档位有质变,而非"伤害线性 +10%"。1/2/3/4/5 层不同梯度跳跃。

**样本**:SF6 Manon Medal 1-5 级伤害爆炸式升级 / 战双 3-Ping。

---

#### 37. 跨回合 / 跨 wave 保留资源

资源不在单回合/单战清零,跨副本 wave 继承。鼓励玩家长期规划。

**样本**:SF6 Manon Medal 跨回合保留 / BO3 元层博弈。

---

#### 38. 独家机制 UI 可见化(对手也能看)

把每角色的独家资源做成 HUD 图标,对手/玩家都能观察 → 变成正反两边博弈层。

**样本**:SF6 血条下小图标显示当前层数/印记。

---

#### 39. "被压制方有救":反制工具不只是闪避

boss 压制玩家时,玩家有 Hold/Parry/反击等反制工具,不是只能挨打。

**样本**:DOA 三角系统(Hold 普世反制,挨打方不绝望) / Sekiro 弹反对撞。

---

#### 40. "招式 IP 化命名":不要"普攻 1/2"

招式名字带 IP 感:"百八式·闇拂"、"八稚女"、"花蝶扇"——名字本身就是角色 DNA。

**样本**:KOF 全系 / Tekken Mishima 风神拳 / 鬼泣 Stinger / Judgment Cut。

**反例**:很多手游"普攻 1 / 普攻 2 / 终结技"。

---

#### 41. "动作辨识 5 米外":姿态本身要够独特

角色姿态/招式动画要让 5 米外能认出谁。看一眼姿态就知道角色。

**样本**:VF Lion 螳螂拳怪异架势 / Akira 八极下沉感 / Taka 相扑推地 / 黑神话棍法三态。

---

#### 42. 行为克制 > 属性克制

不要做纯属性克制(火克草),要做"行为克制":闪避克普攻、弹反克闪避攻击、投技/抓取克弹反、普攻克投技——比石头剪刀布更深。

**样本**:DOA 三角系统(击打/投技/Hold)/ 永劫无间(连击/蓄力/振刀)/ 只狼危技(刺/横/抓)。

---

### K. 设计意图与机制经济(规则 43-46)

#### 43. 操作核心方向先对齐,再展开机制

一个角色"想怎么玩"(核心招放普攻还是特殊技 / 武器怎么玩 / 资源怎么循环 / 配队怎么协同)是**设计意图**,属于只有项目 / 策划知道的信息。**操作核心没先对齐就开猜,容易迭代多轮返工**——机制铺到一半发现核心方向理解错了,整套招式集要推倒重来。

**设计要点**:Phase 1 收集设计输入时,把"操作核心方向"作为**必先对齐项**问清楚,再下沉机制到各招式;别拿"看起来合理的玩法"当默认就一路展开。

**反例**:把核心输出默认放普攻去铺整套连段,展开到大招才被告知"这角色核心是特殊技蓄力,普攻只是过渡"——前面全返工。

---

#### 44. 消极动作不触发爆发(松手 / 停手别当扳机)

用"松手 / 停手"这类**消极动作**触发爆发是反直觉的——玩家的直觉是"主动按键 = 放大招",而"停下来 = 收手";把爆发绑在松手上,玩家会**误触 / 错过时机**,且爽感断点。

**设计要点**:爆发应走**满层自动触发**(攒够条件系统自己炸)或**独立的主动输入**(专门一个键 / 一个长按到顶),让"触发爆发"是一个**积极、可预期**的动作。

**样本**:烬羽的余烬印记叠满 8 层**自动引爆**——命中即炸,不靠玩家松开长按、不需手动按引爆键;玩家的活是"铺得多铺得匀",引爆交给系统判定。

---

#### 45. 资源返现易空转,正反馈走限时窗

消耗型资源若"花了又返"(消耗即回补),循环会**空转**——玩家无脑倾泻资源也不见底,机制失去节流意义。**单向消耗(挣得到 / 花得掉 / 不返还)才有节流**;爆发 / 强化的正反馈不要塞回资源里返现,而应走一扇**限时强化窗**,逼出"攒—花—再攒"的节奏。

**设计要点**:给消耗资源一个**天花板**(单向不返),让正反馈以"限时窗 / 滚雪球窗"形式发放,并用"窗时长上限"夹住,避免一次攒满全程开窗破节奏。

**样本**:烬羽的炉火**单向不返**(普攻 / 闪避攒、引爆绝不返还),引爆的唯一产出是开"余烬窗"(限时强化窗)而非返炉火——逼玩家靠普攻 / 完美闪避把炉火攒回来才有下一波猛铺。

---

#### 46. 异格 / 特别版不换皮:三轴分流

同名角色的"异格 / 特别版 / 复刻版"要在**操作动词 + 定位 + 武器**三轴上做真正的分流,而不是数值 / 外观换皮。只改数值(攻击力 +X%)或只换配色 / 立绘,玩家会立刻识破"这是同一个角色再卖一遍"。

**设计要点**:异格版至少要换其中一到两轴——操作动词换(原版蓄力流 → 异格连段流)/ 定位换(原版主 C → 异格副 C·增益)/ 武器换(刀 → 双枪),让两个版本在**实际操作手感**上分得开。

**反例**:某角色"特别版"只把数值整体上调 + 换一套立绘,招式集 / 操作 / 定位与原版完全一致——社区评价"换皮敛财"。

## 三、机制创新法则 — 从灵感种子推导机制(5 步法)

让 AI 策划**真正出新设计**而非"模仿现有原型"的核心方法。配合 § 一 46 条元规则使用。

### Step 1: 锁定灵感种子(3 类)

| 种子类型 | 例子 | 推导起点 |
|---|---|---|
| **IP 角色种子** | 功夫宗师(武术保镖) / 机甲改装者(钢铁侠+变形金刚气质) | IP 标志性动作 + 性格 + 武器 |
| **武器/工具种子** | 链刃 / 双枪 / 时间能力 | 武器物理特性 + 反直觉用法 |
| **世界观元素种子** | 蒸汽朋克 / 武术宗派 / 妖怪 | 元素的"力学规则" + "美学符号" |

### Step 2: 三问推导核心机制

**Q1: 这个种子最反直觉的"力学规则"是什么?**

不要"火焰角色 = 烧伤 DOT"这种直觉对应。例:
- 火焰种子 → 直觉:DOT。反直觉:火焰**消耗氧气** → 周围生成"无氧区"敌人窒息
- 时间种子 → 直觉:加减速。反直觉:**让敌人和自己时间不同步** → 玩家预判 3 秒前的敌人
- 武术种子 → 直觉:连段。反直觉:武术"含蓄" → 长按防御 Pose 逼对方先动

**Q2: 这个机制怎么贯穿角色的完整招式集?**

把反直觉机制下沉到角色的各招式类型(普攻连段 / 特殊技 / 闪避反击 / 弹反 / 终结技 / 团队联动技等,具体招式集由项目自定义)。每一类招式都应该"有自己版本的这个机制",而不是只有一招体现、其余招式与机制无关。

**Q3: 这个机制能制造什么"新决策点"?**

机制要给玩家**新的决策**,不只是"按出来打高伤"。例如火焰种子无氧区 → 玩家决策"无氧区位置——堵路 / 引敌人进 / 自己在区内会窒息?"

### Step 3: 对照 42 元规则做强度评估

- 命中 1-3 条 → 设计平庸
- 命中 4-7 条 → 不错
- 命中 8 条以上 → 优秀
- 命中 12 条以上 → 出圈级

**命中 < 4 条 → 改设计**,不是"凑数加机制"。

### Step 4: 反模式自检

参见 [反模式自检](anti-patterns.md) 16 条反模式清单。

### Step 5: 命名 = 设计的最后一笔

机制名要**让玩家想用**。
- ❌ "氧气消耗状态"
- ✅ "窒息领域" / "永劫之息" / "无声地带"

命名失败 = 机制再好也没人记得。**预测玩家会取的外号**——这是 § 一 元规则 14 的实现路径。

## 四、设计前的 5 问自检

设计新角色/Boss 前先答这 5 问:

1. **种子是什么**?(IP / 武器 / 世界观)
2. **最反直觉的"力学规则"是什么**?
3. **角色完整招式集的每一类招式如何用这个机制表达**?
4. **新决策点有几个**?玩家"按出来打"以外还要思考什么?
5. **命中几条元规则**?

答完了再开始填战斗设计文档。否则就是套现有原型,不能"举一反三"。
