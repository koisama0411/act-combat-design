## E 型:多体共生 / Turf War 🆕

**适用**:双 boss 战、剧情 boss、双相依存型 boss

### 结构

```
两个 boss 同时存在:
  A 型(主):伤害担当 / 标志性招式
  B 型(辅):护盾担当 / 召唤增援 / 治疗 A
→ 必须先解决 B 才能高效输出 A
→ 或 A/B 共生(分离时虚弱,合体时强力,要在合体瞬间打断)
→ 50% HP 时分化(A 进入愤怒 / B 召唤分身)
→ Turf War 变体:两个独立 boss 互殴,玩家可利用其中一只 KO 另一只
```

### 设计要点

- 两 boss **不同定位**(攻击型 + 辅助型 / 近战型 + 远程型 / 体型大 + 体型小)
- 玩家需要**分配火力**:同打 A/B / 先 B 后 A / 利用环境让两 boss 互伤
- 共生型要有"合体强化"机制——逼玩家在分离瞬间打断

### 样本

- Hades 1 忒修斯 + 阿斯忒里俄斯(双 boss 协同,50% HP 时忒修斯无敌召唤神祇)
- MH Turf War(两怪相遇互砍,玩家可推动它们打架)
- 龙之信条 2 大怪兽 + 小怪护卫
- 黑神话黄风大圣 + 佛头分体

## F 型:环境互动 / 场景战 🆕

**适用**:章节末 boss、史诗级 boss、电影化演出 boss

### 结构

```
boss 攻击不只来自本体,场景本身参战:
  - 场地有"死区"和"安全区",随时间/招式变化
  - 玩家必须站在特定区域才能输出/防御
  - 场景元素可被玩家用作武器/掩护
  - boss 招式改变场景拓扑(Chronos 时钟变形 / Bahamut 轨道射击)
```

### 设计要点

- **场地不是中性的**——它是 boss 的延伸
- 玩家要同时**看 boss + 看场地变化 + 看自己站位**
- 适合做"电影化奇观 boss"(类似猎天使大型 boss)
- 高视觉冲击 + 高难度

### 样本

- Hades 2 Chronos(场地变时钟表面,玩家必须站在数字光圈安全区)
- 黑神话亢金龙(身体盘绕在塔上,玩家爬塔身打飞扑/躲甩尾)
- FF16 Bahamut 战(类似 Panzer Dragoon 的轨道射击战)
- FF16 Titan 战(奥特曼级巨型化,Clive 顺着 Titan 手臂跑上去)

## G 型:教学考核串联 🆕

**适用**:终极 boss、章节末 boss、"集大成"型 boss

### 结构

```
前置 sub-boss 群(教学层):
  sub-boss A → 教「机制 1」(如招架)
  sub-boss B → 教「机制 2」(如盾击)
  sub-boss C → 教「机制 3」(如蓝圈中断)
→ 最终 boss(集大成):
  阶段 1:综合 sub-boss A 的招式
  阶段 2:综合 sub-boss B 的招式
  阶段 3:综合 sub-boss C 的招式
  阶段 4:三者融合的终极考核
```

### 设计要点

- 最终 boss = **本游戏所有核心机制的期末考**
- 前置 sub-boss 是"muscle memory 训练"
- 设计师必须先想清楚"本游戏要教玩家学什么 N 个机制"
- 最大化"成长感"——通关那一刻玩家觉得"我学了一身本事"

### 样本

- 战神 Gna(集前 8 个女武神之大成)
- 只狼苇名一心(四阶段 = 苇名弦一郎 + 真剑圣 + 雷返 + 双段雷枪扫斩)
- MGR 参议员 Armstrong(综合 Blade Mode + 招架 + Zandatsu + 燃料管理)
- Jedi Survivor Dagan Gera(三次出现,逐次升级)
- Hades 1 哈迪斯王(3 阶段,每阶段加新招式叠加)

## H 型:伪 KO 反转 🆕

**适用**:剧情高潮 boss、单个 boss 但要二阶段戏剧反差

### 结构

```
阶段 1:正常 boss 战,玩家打到 HP 见底
→ "假死决胜"过场动画(boss 倒下,玩家以为赢了)
→ 阶段 2 启动:boss 复活 + 整套招式集换掉
  - 武器换(扔斧子变摔角手套 / 捡头变长太刀)
  - 形态换(变形 / 跨形态)
  - 节奏换(慢 boss 变快 / 单挑变群战)
→ 玩家心理预期被打破,重新学一遍
```

### 设计要点

- 反转触发**必须有强视觉提示**(过场动画 / 屏幕变色 / BGM 切换)
- 第二阶段招式**不能是第一阶段的简单升级**——要"完全不同"
- 适合剧情节点 boss(boss 是关键反派,要"震撼")
- 数值上可以单 HP 条,也可以重置 HP

### 样本

- 艾尔登戈弗雷 → Hoarah Loux(一阶段拿斧子,二阶段扔斧子变摔角手套全套抓投)
- 只狼猩猩(伪 KO + 捡头复活变长太刀 + 死亡咆哮异常)
- Sephiroth Reborn(高达脸 + 白色核心,完全新机制)
- 黑神话黄风大圣(一阶段长柄武器,二阶段佛头大盾+三昧神风)

## I 型:机制反向逼配队 🆕

**适用**:深渊关 / 高难度副本 / 终局挑战 boss

### 结构

```
Boss 默认免疫单一玩法:
  - 必须多段攻击才掉血(只有 1 段攻击的角色无效)
  - 必须特定属性组合才能破护盾(单属性队伍卡死)
  - 必须达到 DPS 阈值才能避免群伤秒队
→ 用机制强迫玩家"阵容多样化"
→ 反向驱动养成系统(玩家必须养多角色)
```

### 设计要点

- 这是**养成驱动机制**——让玩家"为打这个 boss 必须养更多角色"
- **必须有"折中方案"**(参见 [反模式自检](anti-patterns.md) 反模式 8),否则会变成"卡角色付费墙"
- 玩家社区反应敏感,平衡设计要小心

### 样本

- **崩 3 量子奇点**:必须多段攻击堆累计才掉血——逼玩家用高频段数角色(深渊弦音、爱衣)
- **MH Alatreon DPS check**:必须在 X 时间内打够元素伤害,否则群伤秒队
- **原神风行迷踪**等特殊机制怪物
- **绝区零某些精英怪**:失衡槽极长,不带"击破型"角色 DPS 腰斩

## K 型:分支 mini-boss 池 🆕

**适用**:挑战副本 / Boss Rush 模式 / 章节末综合 boss

### 结构

```
玩家进入"分支结构"(棋盘 / 转盘 / 迷宫):
  - 必须先 KO N 个 mini-boss(从 M 个池子中随机抽,M > N)
  - 每个 mini-boss 是完整 boss 战(2-5 分钟)
  - 全部 KO 后挑战终 boss(集大成 / 综合 sub-boss 机制)
→ 玩家每次重打都有"不同 mini-boss 组合"
→ 高重玩价值
```

### 设计要点

- **mini-boss 池要 ≥ N + 3**(确保随机性 + 重玩价值)
- 每个 mini-boss 都要"小而精"(招式数 3-5 招 + 1 个独家机制)
- 终 boss 可以综合或独立(Cuphead King Dice 本体相对独立 + 9 mini-boss)
- 适合做"挑战模式" / "无尽塔" / 章节末综合考试

### 样本

- **Cuphead King Dice**:桌面棋盘 + 走骰子格 + 触发 mini-boss(每停一格触发一个) → 走完打 King Dice 本体(扑克牌弹幕 + 骰子魔术 + 召唤)
- **Vampire Survivors Boss Rush 模式**:多个 boss 池随机选
- **杀戮尖塔精英怪节点**:玩家选路径决定打哪些精英(每个池 30+ 精英怪)
- **黑神话章节末"妖怪联军"**:章节内打过的 boss 会在终战前再现

**设计启发**:作为"挑战副本"或"无尽塔"的核心结构——**mini-boss 复用 = 美术成本低 + 重玩价值高**。

## L 型:形态切换爆炸 boss 🆕

**适用**:章节末 boss / 演出强的剧情 boss / Boss Rush 模式

### 结构

```
boss 有 3-4 形态:
  - 每形态完全换模型 + 美术 + 招式池(不是改色调,是"完全变成另一种生物")
  - 形态间无衔接动画,直接切换("我变了")
  - 每形态都是独立的"小关"(招式数 3-5 招/形态,总 12-20 招)
  - 战斗时长 1-3 分钟/形态 → 总 5-10 分钟
```

### 设计要点

- **每形态招式池 ≠ 升级版,而是"完全不同的设计"**
- 招式数克制(≤ 5 招/形态),不堆砌
- 美术工作量大,但能做出"年货级"印象 boss
- **跟 H 型(伪 KO 反转)的区别**:H 型是"1 HP 条 2 形态",L 型是"多 HP 条 N 形态"且**每形态都是完整 boss 战**

### 样本

- **Cuphead 全 boss**:每个 boss 都是 3-4 形态(基础范式)
- **Cuphead The Devil(终 boss)**:4 阶段 5 形态(龙 / 巨头 / 山羊 / 蛇 / 终态),招式池 20+ 招
- **Cuphead Grim Matchstick(龙关)**:3 形态切游戏类型(地面战 → 飞机弹幕段 → 三头蛇追踪)
- **Bayonetta 系列 boss**(白金范式):每形态屏幕级演出 + Climax/QTE 嵌入

**设计启发**:适合做"剧情高潮 boss" — 配合"美术 + 音乐"的"年货级 boss",让玩家"打通后还想再打"。如果项目里有玩家可用的「双形态切换」角色原型,可以反向推成 boss 范式——boss 也能用形态切换扩展戏剧张力。

## J 型:镜像玩家 Build 🆕

**适用**:终局 boss / 终极挑战 / 剧情反派对决

### 结构

```
Boss = 玩家本身的"反派版本":
  - 拥有玩家全部的机制(闪避/弹反/招架/连段)
  - 招式表是玩家招式集的"反向应用"
  - 玩家学会的所有技巧 boss 都会
→ "我打的是另一个我"
→ 终极考验:玩家必须比自己更熟练
```

### 设计要点

- 适合"剧情对手"角色(主角的镜像 / 反派 = 阴影版主角)
- 招式表借用玩家的"招式集"但加强(更快 / 更狠 / 更猛)
- Boss 也会闪避 / 弹反 / 招架玩家的攻击 → 玩家不能"无脑硬刚"
- 战斗本身成为"主题的视觉化"(打自己 = 战胜自己)

### 样本

- **AC6 Ayre**:终局 boss 用接近玩家 build 的范式打,速度极快
- **只狼苇名弦一郎**:用主角风格的太刀打,招式可读但快
- **NieR 9S vs 2B**:剧情驱动的镜像对决
- **DMC5 Vergil**:跟 Dante 招式互为镜像

> 提示:若项目里有玩家可用角色,可以让其 boss 版本与玩家版部分招式互相呼应——既复用资产,又强化"镜像对决"主题。

## 候选未沉淀范式(来自格斗游戏调研)

7 种 boss 范式候选(格斗游戏借鉴):

| 候选 | 代表 | 核心 |
|---|---|---|
| **自损爆发 boss** | Strive Nagoriyuki 模板 | boss 主动扣血加倍输出,逼玩家在 boss 自损期间打爆 |
| **破墙换场景 boss** | Strive Wall Break | boss 阶段切换 = 被打飞到新场景,而非数值阈值 |
| **buddy 共生 boss** | Skullgirls Painwheel | boss 本体+寄生虫,先杀寄生虫降低 boss 招式 |
| **多姿态形态 boss** | VF Lei-Fei 五形 / Tekken Jin Awakened | boss 满足条件进第二/第三形态,招表完全换 |
| **招式借用 boss** | DOA Honoka 模板 | boss 学习玩家用过的招,玩家不能依赖单一套路 |
| **队伍编排 boss** | KOF 3v3 | boss 是 3-5 个分体,顺序击破有差异 |
| **地形机关 boss** | DOA Stage Hazard | boss 招式跟场地互动,场地是 boss 的第三只手 |
