# 打击感技术规范(工程化基线)

⚠️ **本文件性质**:跟其他 references 不同,本文件是**工程化技术规范**,不是"设计思路"——可直接作为你项目的"打击感配置基线"。

业界 16+ 款 ACT/格斗游戏调研沉淀。**核心心法**:二游 ACT 差 3A 一档**不是投入不够**,而是"**全维度偏保守、偏均匀、缺克制**"——敢不敢让重击真的重、敢不敢让关键命中真的突出。

---

## 一、5 大技术柱

### 1. Hit-Stop / 顿帧(命中冻结)

**Sakurai 4 法则**(行业圣经):
1. 攻防双方**对等冻结**,时长完全一样(信息对等)
2. **伤害越高,hit-stop 越长**(线性挂钩)
3. **攻击者也微振**(1-2 像素 visual 振动,双向反馈)
4. **Hurtbox 保持静止**,只振动 visual(防"看上去打到了实际没打到")

**经典参数**:
- 鬼泣 5:轻 3-5f / 重 6-10f / 必杀 10-15f
- 街霸 6:轻 8f / 重 12f / Super Flash 16f(单方能输入)
- 黑神话:重击 5-8f / 棍花完美 8-12f / 法术变身 15-20f
- 二游(鸣潮/绝区零/战双):普遍偏短 2-6f(怕卡顿反失重量)

**设计要点**:
- 重击 hit-stop 必须比轻击长 **50% 以上**(否则失重量感)
- 超过 20f → combo 流畅性割裂(玩家觉得"卡顿"而非"重量")
- 冻结要停在"**最美姿态帧**"——刀划到肉里的那一帧,而不是接触瞬间
- 必杀技用 hit-stop + 慢动作复合(街霸 Super Flash / DMC Devil Trigger)

### 2. Camera Shake / 屏幕震动

**业界参数**:
| 参数 | 范围 |
|---|---|
| 强度(intensity) | 0.1-1.0(轻 0.1-0.2 / 重 0.35-0.55 / 必杀 0.75-1.0+位移) |
| 持续(duration) | 0.05-0.5s(轻 0.05-0.1s / 必杀 0.35-0.6s) |
| 频率 | 8-30 Hz(高频 25-30 = 碎冲击 / 低频 8-12 = 沉重量) |
| 衰减曲线 | **指数衰减**(自然) |

**技术选型**:
- **Perlin Noise** → 高强度震动(必杀/重击),不反复横跳
- **Sinusoidal Wave** → 低强度持续(载具/心跳)
- **Cinemachine Impulse**(Unity)→ 行业默认,支持冲击波传播
- **方向化震动** → 命中方向决定主震动轴,不全方向乱抖

**反模式**:
- 过度震动 → 玩家晕(最常见反例)
- 不分等级全攻击都震 → 重击失去"重"
- 必杀级震动用在常规攻击 → 玩家麻木,真大招反而无感

### 3. 音效 / BGM 联动

**命中音频段分级**(必须做):
| 类型 | 频段 | 时长 | 后处理 |
|---|---|---|---|
| 轻击 | 中高频 2-4kHz 清脆 | 100-200ms | 干声,短混响 |
| 重击 | 中低频 200-800Hz 沉闷 | 300-500ms | 长混响 + Low-pass tail |
| 暴击 | 全频段 + 高频 sweetener | 400-600ms | 短延时叠音 |
| 必杀 | 子低频 50-100Hz 起 + 高频闪光声 | 500ms-1.5s | 反向 reverb + 全频段冲击 |
| 弹反 | 极高频 4-8kHz 金属 + 短 transient | 50-150ms | 干声,压缩重(锐利) |

**BGM 联动机制**:
- **触发式分层**(DMC 5 / Hi-Fi Rush):combo 等级升 → 加层鼓/吉他
- **状态切换**(MGR / 黑神话):boss 二阶段 → 主题歌切副歌;残血 → 紧张层激活
- **节拍锚点**(Sekiro):弹反"叮"声精确落在 BGM 拍点 → 音效成为节奏一部分

---

### 4. VFX / 视觉特效(5 层堆叠)

| 层级 | 元素 | 持续时间 | 作用 |
|---|---|---|---|
| **L1 命中点** | 火花 / 撞击粒子 | 100-200ms | "肉感"瞬时反馈 |
| **L2 光效** | Bloom Flash / 屏幕泛色 | 50-100ms | 强化瞬间冲击 |
| **L3 飘溅** | 碎片 / 血液 / 元素飞溅 | 300-800ms | 物理反馈 |
| **L4 残影** | 武器拖影 / 角色拖影 | 200-400ms | 表现速度 |
| **L5 余韵** ⭐ | 能量场 / 元素附着 / 烟雾 | 500-1500ms | **"重量感"真正来源** |

**设计要点**:
- **错峰出场**(50-150ms 入场差),避免视觉爆炸
- **色彩区分属性**:火红 / 冰蓝 / 雷紫 / 物理白
- **大特效只在关键命中**(暴击/终结技),普攻只用 L1-L2
- **L5 余韵层是手游 ACT 常漏的部分** —— 没 L5 = 失重量感

**反模式**:
- 每一击都全屏闪白 → 麻木
- 特效遮挡角色/boss 判定
- 特效色彩同质化(全金黄)→ 无法区分招式类型
- **L5 余韵缺失** → 命中"轻飘飘"(很多手游通病)

---

### 5. Haptic / 振动反馈(PS5 DualSense)

**业界标杆**:
- **Insomniac 蜘蛛侠 PS5**:方向化触觉(不同方向受击从手柄不同位置振)
- **战神 Ragnarok**:斧召回 haptic 渐强 → 接触手掌瞬间"meaty thud";不同地表(雪/冰/木/金属)haptic 不同
- **Stellar Blade**:弹反时 adaptive trigger 瞬时阻力增加

**设计要点**:
- **方向化** + **材质化** + **节奏化**(必杀分 2-3 段)
- Adaptive Trigger 阻力(弹反/必杀)
- 手机端用线性马达替代(短脉冲 0.05-0.3s,做不到方向化但能做强度分级)

## 三、二游 ACT vs 3A 的 7 大差距

战双/绝区零/鸣潮 vs 鬼泣/黑神话/猎天使的"次一档"是**结构性差距**:

1. **Hit-Stop 偏短偏均匀** — 重击没显著长于轻击 → 失重量
2. **音效写实度不够** — 用"动画式 SFX",缺真实金属/肉体采样夸张化
3. **VFX 堆叠过度** — 关键命中没"突然变猛"层级感
4. **Camera Shake 配比保守** — 必杀"震不动玩家椅子"
5. **缺动态 BGM 系统** — 战斗音乐循环 loop,不跟事件联动
6. **Haptic 投入少** — 手机/PC 主战场,DualSense 是加分项
7. **动画原画质量** — 受成本限制无法人均 mocap + 手 K → 即使技术参数对齐,"姿态本身"不够帅

**结论**:差距不在投入,在"**敢不敢克制 + 敢不敢让重击真的重**"。

## 四、5 大落地优先级(投入产出比从高到低)

### 优先级 1:**Hit-Stop 体系**(成本最低、提升最大)
- 不需要美术资源,改 timeline / 技能配置即可
- 全角色统一标准:轻 3f / 中 5f / 重 7f / 暴击 10f / 必杀 15f
- **关键:重击 hit-stop 必须比轻击长 50%+**
- 投入:1-2 人周 × 工程师 / 战斗策划

### 优先级 2:**音效分层 + 动态 BGM**(中等成本,提升明显)
- 采购或自制真实金属/肉体采样库 → 夸张化处理
- 全角色按"频段分级表"重做 SFX
- BGM 加 1-2 层动态分层(combo 高加鼓 / boss 二阶段切副歌)
- 投入:1-2 个音频工程师 × 1-2 月

### 优先级 3:**Camera Shake 分级 + 关键命中演出**
- Cinemachine Impulse 体系 → 全攻击统一接入
- 必杀/暴击加 zoom-in + 镜头位移(单独 Timeline)
- 投入:1 个工程师 × 2 周搭框架 + 战斗策划逐技能配参

### 优先级 4:**VFX 5 层堆叠规范 + L5 余韵**
- 美术成本最高,但有分级标准后特效美术可"按表配"
- 关键:**L5 后摇余韵层**(很多二游漏掉) + **屏幕泛白**(暴击)
- 投入:特效美术按角色重做命中 VFX,3-5 天/角色

### 优先级 5:**Haptic 适配**(锦上添花)
- PC PS5 用户的"惊喜点"
- 接入 DualSense API + 方向化 + Adaptive Trigger 弹反阻力
- 投入:1 个工程师 × 2-3 周

## 五、14 条反模式(必避)

| # | 反模式 | 后果 |
|---|---|---|
| 1 | Hit-stop 太长(>20f) | combo 卡顿 |
| 2 | Hit-stop 一刀切不分轻重 | 失重量感 |
| 3 | 屏幕震动全攻击都重抖 | 玩家晕 + 重击不重 |
| 4 | 屏幕泛白普攻就用 | 暴击/必杀无视觉差异 |
| 5 | VFX 五层全开普攻 | 关键命中没层级 |
| 6 | VFX 余韵 L5 缺失 | 命中"轻飘飘" |
| 7 | 命中音效单调 | 玩家听几小时疲劳 |
| 8 | BGM 静态不联动 | 战斗无情绪起伏 |
| 9 | Haptic 全方向乱抖 | 浪费 DualSense |
| 10 | Adaptive Trigger 不分技能 | 玩家手指麻 |
| 11 | 判定盒在 hit-stop 时跟着抖 | 看似打中实际没打中(Sakurai 第 3 法则反例) |
| 12 | 攻击者完全不振动(只受击方抖) | 攻防反馈不对等 |
| 13 | 大招级演出用在中等攻击 | 大招演出无升级感 |
| 14 | 不分平台一套参数 | 手机震屏导致玩家手晕 |

## 六、落地步骤(项目内部)

1. **基线对标**:选标杆(如鬼泣 5 Nero / 战双某高质感角色),逐帧拆解 → 量出 hit-stop / shake / VFX 余韵 时长 → 建立项目内部基线表
2. **统一配置体系**:在你项目的技能 / timeline 配置里加 hit-stop / camera shake / vfx / sound 5 个标准字段,每个攻击节点必填
3. **分级模板**:轻/中/重/暴击/必杀 5 个模板,招式按模板选,不是每招从零配
4. **A/B 测试**:同一招式两个参数版本盲测,验证"调到位 vs 不到位"的感知差距
5. **多端适配**:PC / 主机 / 手机 各有配置档(手机 camera shake 减弱 30%,haptic 用线性马达替代)
