# 跨游戏借鉴库(主题索引)

按"设计主题"分组的样本库——遇到具体设计问题时,知道"哪个游戏怎么解决的"。

⚠️ 本文件不是知识完整列表,而是**主题索引** + **代表样本**——设计时可以快速查到借鉴对象。
每个主题里留了「你的项目当前方案:____」占位行,填上你项目的现状,方便对照业界样本找差距。

### 二元(基础)
- 你的项目当前方案:____(示例:某色锁定 = 必闪 + 对撞弹反)

### 三元(进阶)

- **战神 Ragnarök**:黄圈招架 / 红圈必闪 / 蓝圈可中断(盾击)
- **只狼危技**:红色刺(见切) / 红色横(必跳) / 红色抓(必侧滚)

### 四元(顶级)

- **Jedi Survivor**:红必闪 / 黄必躲(抓) / 蓝弹反 / 白偏转(投射)

### 一键化(简化派)
- **卧龙·苍天陨落**:化解一键通吃普攻 + 红光必杀
- **猎天使魔女**:Witch Time(完美闪避 → 子弹时间)

### 极限化(高门槛派)
- **鬼泣 5 Royal Guard**:挨打-1s 松键反击的 0.2s 极限窗口
- **MGR**:招架 = 攻击键 + 朝向(防御即攻击)

### 设计警示
- **卧龙**:化解收益过高 → 其他玩法贬值(玩家社区诟病)

- 你的项目当前方案:____(示例:角色专属蓄力槽 / 能量条)

### 双向槽(攻防一体)

- **Relink**:Link 槽全队 share

## 主题 4:化学反应 / 组合催化

### 属性紊乱(叠加爆炸)

- **FF7 Rebirth Synergy Skills(免费协同) + Synergy Abilities(消耗协同)**:每对角色 3-5 个独家组合
- **Persona 5 Showtime**:特定双人组合触发演出技

### 五行/姿态克制
- **卧龙五行**:金木水火土相克

## 如何使用本文件

1. **遇到具体设计问题**(如"怎么做配队 Synergy?") → 查对应主题章节
2. **从样本反推机制**:看到一个游戏的机制,在本文件查它属于哪个主题,对照同主题其他样本找异同
3. **填占位行**:把每个主题的「你的项目当前方案:____」填上,方便对照业界样本找差距
4. **更新本文件**:调研新游戏时,新机制要补到对应主题章节(或新建主题)
5. **遵循"主题索引 + 代表样本"原则**:不堆完整描述,只列"关键词 + 1-2 句辨识特征"
