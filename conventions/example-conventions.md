# 示例项目战斗约定（已填写范例）· Example Filled Conventions

> 这是 [TEMPLATE.md](TEMPLATE.md) 填好后长什么样的**范例**。
> 它属于一个**虚构的示例项目**——单机 2.5D 动作 RPG，代号 **《孤刃行》(LoneBlade，虚构)**。配套示例角色「[苍岚](../output/characters/cang-lan/md/苍岚-战斗设计.md)」就是在这套约定下设计的。
>
> ⚠️ **注意它和别的项目可以完全不同**：这里是 60fps、精简招式集、金紫双色预警、买断制无抽卡、单角色主导——刻意区别于"二游三人队 + 抽卡命座"那类项目，用来证明本方法论**不绑定任何一套体系**。你自己的项目按 TEMPLATE 重填即可。

## 1. 项目基调与视角
- 视角 / 维度：**2.5D**（3D 角色模型 + 半固定第三人称镜头，主要在水平面走位，有限纵深）
- 距离单位：**米（m）**，世界单位 1 = 1m
- 帧率基准：**60 fps**（所有前摇 / 后摇帧数按 60fps 计）

## 2. 招式集（必备）
每个角色**必备**：
1. **普攻连段**（3–4 段，起手轻、末段重，末段常带击退 / 浮空）
2. **闪避**（步伐 / 翻滚，带"完美闪避"窗 6 帧）
3. **核心特殊技**（角色机制载体，1 个）
4. **终结技 / 大招**（能量满释放，1 个）
5. **格挡 / 弹反**（可格挡金色预警，完美时机触发弹反反击）

**可选 / 角色特色**：空中攻击、处决（对倒地 / 破韧目标）。

**衔接 / 取消规则**：
- 普攻任意段可被**闪避**取消
- 蓄力类技能可被**完美闪避**取消，并保留已积累的资源
- 终结技释放期间**霸体**，不可被打断

## 3. 角色分类轴
- **轴 1 · 武器型**：单刀 / 长柄 / 拳套 / 远械
- **轴 2 · 队伍定位**：输出 / 坦克 / 支援
- **无稀有度**（买断制，所有角色平等可练）
- **上手难度**：★1–3

## 4. 视觉预警契约
| 预警 | 含义 | 应对 |
|---|---|---|
| **金色闪光** | 可格挡攻击 | 格挡 / 完美时机弹反 |
| **紫色闪光** | 必闪攻击（无法格挡） | 必须闪避 |
| 无闪光 | 普通攻击 | 自由应对 |

规则：预警光效必须**覆盖整个前摇**，紫色必闪招前摇 ≥ 18 帧给足反应窗。

## 5. 受击 / CC 状态分类
| 状态 | 持续 | 衔接 |
|---|---|---|
| **硬直** | 极短(~0.15s) | 原地踉跄，可被普攻继续 |
| **击退** | 短(~0.3s) | 水平后退不离地 |
| **浮空** | 中(~0.8s) | 离地，可接浮空连段；二次击中可"持续浮空" |
| **倒地** | 状态(~1.5s) | 需起身；可被"处决"触发 |

破韧（韧性条耗尽）→ 强制**倒地 + 3s 易伤窗**。

## 6. 命名规范
- 动作（PascalCase）：`Attack_01`…`Attack_04` / `Skill_Charge_Pre·Loop·Release·Recover` / `Dodge` / `Parry_Start·Success·Attack` / `JumpAttack` / `Ultimate_01`
- 特效：`Vfx_<角色>_<招式>_<段位>`（如 `Vfx_CangLan_Skill_Release_Lv4`）

## 7. 团队 / 配队模型
- **队伍规模**：单角色主导，可选最多 3 人小队（单人能通关，组队是增益不是门槛）
- **协同机制**：纯属性叠加 + **切人联动技**（切换角色瞬间触发一次联动攻击）；**无**共享资源槽、**无**驻场延续
- **主轴 / 出伤思路**：当前操作角色为核心出伤点，队友提供切人爆发 + 被动光环

## 8. 养成 / 成长模型
- **买断单机**：技能树（解锁 / 强化招式）+ 装备词条
- **无抽卡、无命座**（所以角色机制一次给全，不留"必抽满命才完整"的设计）

## 9. 数值 / 伤害框架
- 伤害公式架构：**乘区式**（基础伤害 × 增伤区 × 易伤区 × 暴击区）
- 击破 / 韧性系统：**有**（破韧 → 倒地 + 易伤窗）
- buff 叠加：同名取最高，异名相加

## 10. 可选状态 / 元素轴
- **无全局元素系统**。角色各自的资源（如苍岚的「势」）是角色自带机制，不是跨角色元素轴。
