# 示例项目战斗约定（gacha 范例）·《星澜纪》

> 这是 [TEMPLATE.md](TEMPLATE.md) 填好后的**第二个范例** —— 一个**虚构的抽卡 / gacha 二游**，代号 **《星澜纪》(StarTide，虚构)**：3D 三人队 + 抽卡命座 + 养成物套装 + PVP。
>
> ⚠️ 它**刻意区别于** [《孤刃行》（买断制单机）](example-conventions.md)，用来证明同一套方法论也能落到「二游三人队 + 命座 + 养成物」这类项目。

## 1. 项目基调与视角
- 视角 / 维度：**3D 第三人称**（完整平面走位 + 纵深）
- 距离单位：米（m）
- 帧率基准：**60 fps**

## 2. 招式集（必备）
每角色**必备**：1) 普攻连段（4–5 段）2) 闪避（含完美闪避窗）3) **核心特殊技**（机制载体）4) **终结技 / 大招**（能量满释放）5) **切人联动技**（切换上场触发一次）。**可选**：空中攻击、处决、弹反。

## 3. 角色分类轴
- **武器型**：单刀 / 长柄 / 拳套 / 远械 / 法器
- **队伍定位**：主 C / 副 C / 辅助 / 击破
- **稀有度**：A / S / SSR
- **元素**：五行（金 / 木 / 水 / 火 / 土），每元素带 增伤% / 抗性 / 弱点增伤
- **上手难度**：★1–3

## 4. 视觉预警契约
| 预警 | 含义 | 应对 |
|---|---|---|
| **红光** | 必闪攻击 | 闪避 |
| **黄光** | 可弹反攻击 | 弹反 / 格挡 |
| 无 | 普通攻击 | 自由应对 |

## 5. 受击 / CC 状态分类
硬直 / 击退 / 浮空 / 击飞 / 倒地；**破韧**（韧性条耗尽）→ 强制倒地 + 易伤窗。

## 6. 命名规范
- 动作（PascalCase）：`Attack_01` / `Skill_Charge_*` / `Dodge` / `Parry_*` / `Assist_01` / `Ultimate_01`
- 特效：`Vfx_<角色>_<招式>_<段位>`

## 7. 团队 / 配队模型
- **队伍规模**：三人队（一主双辅）
- **协同机制**：切人联动技 + 后台挂状态 + 共享「协力能量」槽
- **主轴 / 出伤思路**：主 C 核心出伤，双辅助服务其爆发窗

## 8. 养成 / 成长模型（gacha）
- **抽卡 gacha**：等级 + **命座 C0–C6**（每命座解锁 / 强化效果）+ **养成物套装**（圣痕类，2 / 4 件套）
- 命座按 **5 段哲学**：C0 必保完整 / C2 质变拐点 / C6 机制变质（见 [numbers-and-progression.md § 五](../references/numbers-and-progression.md)）
- 有 **PVP**：满命 / 0 命强度差需控（见 § 五 PVP 阈值）

## 9. 数值 / 伤害框架
- **乘区式**：`技能 × 增伤 × 暴击 × 防御 × 抗性`；**增伤为单一加法区**（自拐会被面板稀释，见 [§ 四](../references/numbers-and-progression.md)）
- **击破 / 韧性**：有（破韧 → 倒地 + 易伤窗）
- **buff 叠加**：同名取最高、异名相加
- **暴击**：双爆 CR : CD = 1 : 2

## 10. 可选状态 / 元素轴
- **五行元素**：金木水火土，元素弱点增伤 + 抗性穿透；可做元素队协同。
